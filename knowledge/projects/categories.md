# BST Active Projects (canonical list)

Authoritative list of active BST projects grouped by category. Maintained by hand.
Anything in Asana not on this list is treated as archived/stale or a new arrival
needing human triage. Phase-1 budget/invoicing work starts with `contract` only.

Format: one line per project. Lines starting with `#` or `##` are category headers,
blank lines ignored. Codes in brackets are the canonical project code used to name
the project-state JSON file (e.g., `[350-4]` → `knowledge/project_state/350-4.json`).
Projects without a bracketed code (like `SOCOM S0-AD`) use a slugified name.
