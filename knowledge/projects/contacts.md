## Project Notes
Contacts contains commercial, federal Agency, and university contacts.  If the contact is connected to a project, either active or closed, they are connected to that project as well. 

Project Specific information should stay within the Project, not with the Contact. 

Use the Asana add-in for Gmail to send emails to Contacts (look up the name as the task - see below)
