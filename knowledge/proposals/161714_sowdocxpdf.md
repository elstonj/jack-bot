## Executive Summary
This six-month project aims to integrate drone-collected data into the WRF-SFIRE wildfire modeling and forecasting system. Colorado State University's CIRA will lead the effort in collaboration with Black Swift Technologies to develop data ingestion capabilities and evaluate how drone data improves wildfire forecasting accuracy.

## Technical Approach
- Develop drone data ingestion capabilities to integrate with WRF-SFIRE modeling system
- Create a cloud-based data pipeline in collaboration with Black Swift
- Conduct model calibration and validation using drone-collected data
- Run forecasts with and without drone data inputs for comparison
- Perform sensitivity analysis using different types of drone data
- Preliminary model validation exercises

## Products & Capabilities Described
**BST Role:** 
- Black Swift Technologies will collaborate with CIRA to develop and refine the cloud-based data pipeline for drone data ingestion into WRF-SFIRE
- Black Swift will provide drone data collection capabilities (specific drone model/system not detailed in this excerpt)

**WRF-SFIRE System:**
- A coupled atmospheric-fire modeling system for wildfire forecasting
- Will be enhanced to accept drone-collected observational data as inputs

## Use Cases & Applications
- **Primary Application:** Wildfire detection and monitoring
- **Modeling Focus:** Improving WRF-SFIRE forecast accuracy for wildfire behavior prediction through real-time or near-real-time drone observations
