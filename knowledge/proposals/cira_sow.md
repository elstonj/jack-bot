## Executive Summary
This SOW outlines a collaborative project between CIRA and Black Swift Technologies to integrate drone data collection with WRF-SFIRE (Weather Research and Forecasting - Sfire model) for autonomous wildfire detection and monitoring. The goal is to develop a cloud-based data pipeline that enables proactive fire management in rural communities.

## Technical Approach
- Integration of S3 UAS (drone) data with WRF-SFIRE fire behavior modeling system
- Development of cloud-based data pipeline to ingest drone data into WRF-SFIRE
- Model calibration and validation using drone-collected data
- Comparative WRF-SFIRE forecasting runs with and without drone data
- Sensitivity analysis using different types of drone data
- Preliminary model validation exercises
