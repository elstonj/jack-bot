## Executive Summary
This is a budget justification for iMet (a subcontractor to Black Swift Technologies) supporting a NOAA SBIR Phase I proposal. iMet is providing sensor integration, calibration development, and transmitter design work to support the integration of a sensor suite with BST's multi-hole probe platform.

## Technical Approach
iMet's proposed work includes:
- Integration of environmental sensor suite with BST's multi-hole probe
- Development of calibration methodologies and procedures for sensors
- Design of test fixtures for calibration validation
- Design of transmit interface and transmitter specifications for data transmission
- Provision of sensor recommendations and air-deployed sonde recommendations (at no charge)

## Products & Capabilities Described

**BST Multi-Hole Probe**
- An atmospheric measurement instrument
- Intended to be integrated with an iMet sensor suite
- Will include a transmit interface for data transmission

## Use Cases & Applications
- Atmospheric/meteorological measurements via air-deployed platforms
- Integration with sonde systems for data collection and transmission
