## Key Features
- **Programmability**: Fully programmable via Castle Link USB adapter with Windows software
- **Auxiliary Wire**: User-programmable white auxiliary wire for multiple functions including helicopter governor adjustments, audible beacon, RPM output for gyros, ESC arming lock, or receiver arming lock
- **Data Logging**: Available via Castle Link USB adapter; allows reading, analyzing, and storing recorded ESC data
- **Telemetry**: Monitors battery voltage, ESC temperature, and other parameters; requires additional Castle Telemetry Link hardware
- **Protection Features**: Arm Lock Key and RC Arm Lock capabilities
- **Power System Alerts**: Motor tones and LED flashing patterns for over-temperature, excessive load, and locked rotor conditions
- **Firmware Updates**: Free access to all firmware updates via Castle Link software

## Technical Specifications
- Minimum input voltage: 2S
- Maximum input voltage: 8S (33.6V)
- Castle Link compatible with Windows Vista, 7, 8, and 10 (Macintosh compatible via virtualization)
- Compatible with both Futaba S.BUS2 and Spektrum X-BUS telemetry systems
