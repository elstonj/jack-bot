## Executive Summary
This is a blank template form for asserting intellectual property and data rights restrictions on technical data and computer software to be delivered under a DAF SBIR contract. The form requires identification of restricted deliverables, basis for restrictions, asserted rights categories, and signatory information.

## Document Purpose
Form used to identify and assert restrictions on:
- Technical data (excluding computer software documentation)
- Computer software
- Computer software documentation

Restricts items/components/processes developed at private expense.

## Key Form Fields

**Column 1 - Technical Data or Computer Software to be Furnished with Restrictions**
- Identify deliverable technical data and associated items, components, or processes
- For software: identify the software or documentation
- Include applicable CLIN (Contract Line Item Number) for each deliverable

**Column 2 - Basis for Assertion**
- Development at private expense (exclusive or partial) is primary basis
- For technical data: refers to development of the item/component/process
- For software: refers to software development itself
- Must indicate if development was exclusive or partial private expense
- If not private expense, specify alternative basis

**Column 3 - Asserted Rights Category**
- Examples: government purpose license rights from prior contracts
- Rights in SBIR data from other contracts
- Limited, restricted, or government purpose rights
- Specially negotiated licenses

**Column 4 - Name of Person Asserting Restrictions**
- Corporation, individual, or other entity
- Signature required
- Print name and title required
