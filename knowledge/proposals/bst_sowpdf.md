# BST Statement of Work - NSF CO WY Climate Engine Wildfire Project

## Document Metadata
- Type: Statement of Work (SOW)
- Client/Agency: Cooperative Institute for Research in the Atmosphere (CIRA), Colorado State University
- Program/Solicitation: NSF CO WY Climate Engine / Wildfire project
- Date: August 26, 2024
- BST Products/Systems Referenced: S2 (aircraft), BST Soil Moisture Sensor, NOAA NightFOX payload
- Key Personnel: Dr. Kyle Hilburn (CIRA); Beck Cotter (BST, last editor)

## Executive Summary
BST will collaborate with CIRA throughout 2025 to support a wildfire research project by providing soil moisture and vegetation data, converting satellite hotspot and sensor data for use in fire modeling, and conducting rapid deployment of an S2 aircraft equipped with NOAA's NightFOX payload to monitor active fire conditions.

## Technical Approach
- Convert NOAA NightFOX hotspot detection data and BST Soil Moisture Sensor data into formats compatible with the WRF-SFIRE fire behavior model
- Collect soil moisture and NDVI (Normalized Difference Vegetation Index) data over designated areas to assess fuel loading conditions
- Deploy S2 aircraft equipped with NOAA NightFOX payload on rapid-response basis, with deployment timing guided by satellite-based lightning detection data
- Support collaborative activities including meetings and reporting

## Products & Capabilities Described

**S2 Aircraft**
- Rapid-deployable platform for wildfire monitoring
- Can be equipped with NOAA NightFOX payload for real-time hotspot detection

**BST Soil Moisture Sensor**
- Provides soil moisture measurements relevant to wildfire fuel conditions
- Data to be integrated into WRF-SFIRE modeling framework

**NOAA NightFOX Payload**
- Detects fire hotspots
- Will be integrated aboard S2 for airborne deployment

## Use Cases & Applications
- Active wildfire monitoring and response in Colorado/Wyoming region
- Fire behavior prediction and modeling via WRF-SFIRE
- Fuel loading assessment through soil moisture and NDVI measurements
- Lightning-triggered rapid response deployment

## Notable Details
- Project timeline: Throughout 2025
- Integration of multiple data sources (satellite-based lightning, hotspot detection, environmental sensors) to support operational wildfire response
- Demonstrates BST's capacity for rapid deployment in support of real-time natural hazard monitoring
