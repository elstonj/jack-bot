# BlackSwift-USC Proposal for Soil Moisture In-Situ Data Products

## Document Metadata
- Type: Subcontract Proposal / Statement of Work
- Client/Agency: Black Swift Technologies (BST); University of Southern California (USC)
- Program/Solicitation: NASA SBIR Phase II (Soil Moisture topic)
- Date: May 12-14, 2015
- BST Products/Systems Referenced: L-band radiometer mounted on Uninhabited Aerial System (UAS)
- Key Personnel: Dr. Maciej Stachura (BST), Mahta Moghaddam (USC PI), Ben Holstein (USC Contracts & Grants)

## Executive Summary
USC proposes to support Black Swift Technologies' NASA SBIR Phase II project by providing in-situ soil moisture measurements from the SoilSCAPE wireless sensor network during flight testing of BST's L-band radiometer on a UAS near Canton, Oklahoma. USC will maintain, calibrate, and operate 10-20 soil moisture sensor nodes to provide ground-truth validation data for the radiometer system over a one-month period (May 15 - June 14, 2015).

## Technical Approach
- Deploy and operate the SoilSCAPE wireless sensor network (previously developed under NASA funding) at a 300m × 300m test field near Canton, Oklahoma
- Provide in-situ soil moisture measurements at three depths: ~5 cm, ~10 cm, and ~30 cm
- Install sensors at 10-20 locations across the field
- Conduct pre-flight field visits to ensure all wireless sensor network nodes are operational and producing quality data
- Perform maintenance and repairs on sensor nodes as needed
- Transmit real-time data from field gateway to USC
- Coordinate data availability with BST starting late May 2015

## Products & Capabilities Described

**SoilSCAPE Wireless Sensor Network**
- What it is: A wireless sensor network for automated soil moisture measurement developed by USC with prior NASA funding
- Use in this project: Ground-truth validation network for radiometer calibration and accuracy assessment
- Capabilities: Multi-depth soil moisture sensing (3 depths), wireless data transmission, network gateway at field site

**BST L-Band Radiometer on UAS**
- What it is: Microwave radiometer payload designed for airborne soil moisture remote sensing
- Use: Primary remote sensing instrument being flight-tested at the Canton, Oklahoma test site
- Ground-truth requirement: In-situ data from SoilSCAPE network needed to validate radiometer measurements

## Use Cases & Applications
- **Soil Moisture Remote Sensing Validation**: Flight testing L-band radiometer for soil moisture estimation via microwave radiometry
- **Field Campaign Location**: Canton, Oklahoma test site
- **Test Domain**: Approximately 300m × 300m field area with 10-20 measurement locations
