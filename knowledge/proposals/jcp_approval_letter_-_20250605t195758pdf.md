# JCP Approval Letter

## Document Metadata
- Type: Certification approval letter
- Client/Agency: Defense Logistics Agency (DLA), U.S./Canada Joint Certification Program (JCP) Office
- Program/Solicitation: Joint Certification Program (JCP) for Militarily Critical Technical Data
- Date: June 5, 2025
- BST Products/Systems Referenced: None specifically named
- Key Personnel: Jack S. Elston (Black Swift Technologies LLC)

## Executive Summary
Black Swift Technologies LLC has been approved for certification under the U.S./Canada Joint Certification Program (JCP) to handle Unclassified DoD Export Controlled Militarily Critical Technical Data. The certification is valid for five years and assigns BST a CAGE code (6PGF9) for accessing controlled defense information through authorized DLA systems.

## Certification Details
- **Certification Type:** JCP Certification for handling Unclassified DoD Export Controlled Militarily Critical Technical Data ("Controlled Technical Information" and "Covered Defense Information")
- **Certification Number:** Located in Block 7b of attached DD Form 2345
- **Validity Period:** Five years from approval date (expiration date in Block 7c of DD Form 2345)
- **CAGE Code:** 6PGF9
- **Effective Date:** June 5, 2025

## Compliance Requirements
- Must maintain current NIST SP 800-171 Assessment on Supplier Performance Risk System (SPRS)
- Must register with and maintain access to:
  - SAM.gov
  - Defense Technical Information Center (DTIC)
  - DLA Internet Bid Board System (DIBBS) with DLA Enhanced Validation (DEV)
- Must submit revised DD Form 2345 if company information changes (name, Data Custodian, address)
- Must NOT provide militarily critical technical data to non-certified entities (violation may result in certification revocation and legal action)

## System Access Information
- 24-72 hours required for DLA systems to update permission settings
- Support contacts provided for system issues, password resets, and data access problems

## Notable Details
- This certification enables BST to access and handle controlled DoD technical data, indicating the company works on defense-related projects requiring export control compliance
- Certification is conditional on maintaining updated NIST SP 800-171 compliance documentation
- BST must retain approved DD Form 2345 for future use and reference
