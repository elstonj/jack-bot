- Date: December 5, 2017
- BST Products/Systems Referenced: S2
- Key Personnel: Dr. Jack Elston (BST), Dan Connors (proposed subcontractor)

## Executive Summary
Dan Connors, PhD, expresses commitment to join Black Swift Technologies as a subcontractor for a NASA SBIR Phase II project focused on developing autonomous onboard intelligence capabilities for UAS. Connors will contribute expertise in computer vision, machine learning, and embedded systems to enhance UAS robustness and enable integration into the National Airspace System.

## Technical Approach
Connors proposes to combine computer vision and machine learning algorithms applied to image and fault datasets to automate feature classification using onboard computing platforms. Specific work includes:
- Development of machine learning framework for identification of faults and required maintenance
- Development and implementation of safe landing zone identification using visual cues
- Safe landing zone identification simulation incorporating a priori wind data and photos from previous BST flights
- Implementation on Nvidia Tegra processor for integration onboard S2 UAS
- Flight data analysis and algorithm performance evaluation post-flight testing

## Products & Capabilities Described
**S2 UAS**
- Target platform for integration of safe landing zone identification system
- Will be equipped with Nvidia Tegra onboard computing for autonomous decision-making
- Proposed for Phase II flight testing of landing zone algorithms
