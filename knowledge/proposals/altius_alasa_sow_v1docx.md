# Altius ALASA SOW v1

## Document Metadata
- Type: Statement of Work (SOW)
- Client/Agency: DARPA
- Program/Solicitation: ALASA (Air-Launched Space Access) kickoff mid-April 2012
- Date: June 2–3, 2012
- BST Products/Systems Referenced: Avionics, GN&C (Guidance, Navigation & Control), ground control systems, Flight Termination System integration
- Key Personnel: Black Swift Technologies (subcontractor role identified)

## Executive Summary
Altius Space Machines proposes development and subscale demonstration of the Boomerang Air-Launched Partially-Reusable Microsatellite Launch System. The effort spans from August 2012 through October 2013, with three phases: conceptual design of rocket and carrier plane (Base Option), medium-fidelity subscale gamma maneuver flight demonstration (Option A), and associated analytical/CFD work. Black Swift Technologies is identified as the avionics and GN&C subcontractor responsible for flight and ground control systems.

## Technical Approach

**Phase I Base Option (3 months: Aug–Oct 2012)**
- Task 1: Gather system-level requirements from DARPA with Altius to establish physical design impacts
- Task 2: Rocket conceptual design trade study using vendor inputs (Ventions upper stage, XCOR engines/pumps), MATLAB/Simulink 3DOF vehicle simulator for trajectory optimization, mass/performance modeling including extendable/retractable nozzle analysis
- Task 3: Carrier plane trade study evaluating White Knight 1&2 (Scaled Composites/Virgin Galactic), Aurora Hauler UAV-modified cargo aircraft; discussions on modifications, operations (piloted vs. unmanned), and cost estimates

**Phase I Option A (13 months: Sept 2012–Sept 2013)**
- Task 4: Derive subscale demonstration system requirements based on full-system concepts from Base Option
- Task 5: Design subscale systems:
  - Subtask 5.1 (Able): Single-stage pressure-fed subscale rocket with parachute recovery (simplified vs. full two-stage pump-fed design), similar outer-moldline to conceptual design
  - Subtask 5.2 (RECUV): Subscale carrier UAV based on selected full-scale carrier plane design
  - **Subtask 5.3 (Black Swift): Avionics, instrumentation, actuation control, communications for rocket and UAV; GN&C autopilot design; test flight trajectory planning; ground control station development; Flight Termination System integration with Edwards AFB test range**
  - Subtask 5.4 (Altius): Separation mechanism design; CAD/dynamic motion models; stress analysis; proof-of-concept testing
  - Subtask 5.5 (All): Detailed test plans, hazard analysis, telemetry compatibility verification
- Task 6: Analytical models for separation dynamics and gamma maneuver:
  - Subtask 6.1: Physics-based separation dynamics model
  - Subtask 6.2: Gamma maneuver and separation CFD models (Altius and Air Force Academy)
  - Subtask 6.3: Wind-tunnel testing with cold-gas rocket plume (Altius and Air Force Academy)
- Task 7: Fabrication and testing of all flight hardware and software (6 months: Mar–Aug 2013)
  - Subtask 7.3 & 7.4 (Black Swift): Avionics, communications, ground control hardware fabrication/integration/test; flight control software development/integration/test
- Task 8: Flight demonstration test program (4 months: Jul–Oct 2013)
  - Subtasks 8.1–8.4: Test plan finalization, captive carry tests, hot captive carry tests, gamma maneuver/separation tests, data analysis

## Products & Capabilities Described

**Black Swift Technologies Role:**
- **Avionics System**: Design of avionics, instrumentation, actuation system control, communication systems for both Carrier UAV and Rocket
- **GN&C (Guidance, Navigation & Control)**: Autopilot design, test flight trajectory planning, guidance/navigation/avionics algorithms for UAV and Rocket
- **Ground Control Systems**: Ground station development and control software
- **Flight Termination System**: Coordination with Edwards AFB test range for FTS integration into UAV control system
- **Integration**: Works with Able (rocket) and RECUV (UAV) on hardware integration into respective flight vehicles

**Specifications/Performance Claims:**
- GN&C must handle vehicle separation dynamics and gamma maneuver flight path control
- Must support both subscale (medium-fidelity) and eventual full-scale system architectures
- Must interface with Edwards AFB test range Flight Termination System

## Use Cases & Applications
- **Gamma Maneuver Demonstration**: Subscale flight test to validate the "gamma maneuver" rocket-assisted zoom climb—identified as the single largest risk in the proposed launch system
- **Data Collection**: Aerodynamic data, loads/stresses data, plume impingement data, separation dynamics validation
- **Subscale Vehicle Testing**: Medium-fidelity UAV and rocket models to retire gamma maneuver risk before full-scale development
