## Project 1: Sensor Fusion and Algorithms for Mapping at Low Altitude Above Rugged Terrain

**Award Details:**
- Agency: NASA
- Program: SBIR Phase I
- Award Date: 07/26/2018
- Contract #: 80NSSC18P1915
- Topic/Subtopic: A2.02
- Commercialization Status: Follow-On Sales

**Commercial Outcome:**
Contract with IPOZ Systems LLC totaling $46,088 (3-phase project, $20,688 paid for Phase 1) for UAS-type imaging system for underwater applications.

**Technology Used:**
- Global shutter block camera
- TX2 processing board
- IMU integration

**Deliverables/Capabilities:**
1. Integrated camera and single board computer (SBC) solution for underwater applications
2. Image capture at 1 Hz (adjustable up to 30 Hz theoretical maximum)
3. Timing characterization from trigger to image capture with metadata inclusion
4. CSV logging of image names and IMU time
5. Ethernet-accessible image storage
6. 1TB data storage capacity
7. Powerful SBC enabling future machine learning
8. Integration with GIPSEA software
9. Coverage maps with live thumbnails for near real-time QC
10. Recording start/stop capability
11. Post-Phase I development of stereo depth sensor for collision avoidance feedback to ROV operators
