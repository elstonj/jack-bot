## Executive Summary
This is a comprehensive instructional manual from the USDA National Institute of Food and Agriculture (NIFA) providing step-by-step guidance for organizations and individuals preparing and submitting grant applications through Grants.gov. The document covers registration requirements, form completion, submission procedures, and compliance standards for NIFA funding opportunities.

## Technical Approach
The guide provides procedural instructions for:
- Organization registration and Central Contractor Registry (CCR) / System for Award Management (SAM) requirements
- Software requirements (Adobe Reader compatibility)
- Navigation and completion of standard federal forms (SF-424 R&R and related forms)
- Attachment formatting standards (.pdf format, 12-point font minimum, 1-inch margins, file name restrictions)
- Application submission via Grants.gov portal
- Post-submission tracking and status verification

## Notable Details

**Application Requirements:**
- All attachments must be in PDF format (non-negotiable compliance requirement)
- File names limited to 50 characters, no special characters, spaces, or periods
- Font requirements: minimum 12-point, with 1-inch margins
- Sequential page numbering required
- PDFs cannot be password-protected

**Submission Procedures:**
- Applications considered on time if submitted before 5:00 p.m. ET on deadline date
- Only Authorized Representative (AR) can sign and submit applications
- Grants.gov issues confirmation emails within 2 days (receipt confirmation, then validation status)
- Late applications only considered under extenuating circumstances with documentation

**Forms Required:**
- SF-424 (R&R) Cover Sheet
- R&R Project/Performance Site Location(s)
- R&R Other Project Information
- R&R Senior/Key Person Profile (Expanded)
- R&R Personal Data
- R&R Budget
- R&R Subaward Budget Attachment (optional)
- Supplemental Information Form
- AFRI Project Type Form (if applicable)
- SBIR/STTR Information (if applicable)

**Key Compliance Points:**
- Active CCR/SAM registration required at time of submission
- DUNS number mandatory in all applications
- Organization must maintain current registration while application is under consideration
- Applications submitted after deadline may be rejected unless extenuating circumstances documented
- Duplicate applications submitted concurrently will be excluded from consideration

**Support Contacts:**
