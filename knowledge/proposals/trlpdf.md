## Executive Summary
This document is a standard reference appendix defining the DoD Technology Readiness Level (TRL) scale from levels 1-9. It provides authoritative definitions and descriptions for each TRL stage, serving as a baseline framework for technology maturity assessment in the DARPA-ERIS proposal submission.

## Content Overview
This is not a BST proposal or capability document, but rather a **standardized DoD/DAG (Defense Acquisition Guidebook) reference table** included as an appendix to support TRL discussions in the DARPA-ERIS submission package.

### TRL Scale Summary
| Level | Definition | Stage |
|-------|-----------|-------|
| 1 | Basic principles observed and reported | Research begins |
| 2 | Technology concept/application formulated | Invention phase |
| 3 | Analytical and experimental proof of concept | Active R&D initiated |
| 4 | Component/breadboard validation in lab | Integration begins |
| 5 | Component/breadboard validation in relevant environment | Simulated conditions |
| 6 | System/subsystem prototype in relevant environment | Major readiness increase |
| 7 | System prototype in operational environment | Operational demonstration |
| 8 | Actual system completed and qualified | Final form testing |
| 9 | Actual system proven through successful mission ops | Operational deployment |
