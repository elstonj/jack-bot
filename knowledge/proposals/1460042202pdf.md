## Executive Summary
Black Swift Technologies developed a high-rate compact data logger capable of sampling at 60 megasamples per second (MS/s) for integration into small Unmanned Aircraft Systems (sUAS). The system combines highly accurate differential analog input with solid-state storage and optional integration with the Black Swift SwiftPilot autopilot for soil moisture mapping applications.

## Technical Approach

**System Architecture:**
- **Analog/RF Board**: Interfaces with instruments and auxiliary sensors; conditions analog signals
- **A/D Board**: Utilizes Linear Technology LTC2143-14 (14-Bit, 80MS/s, Low Power Dual ADC)
- **FPGA Module**: Spartan-6 based EFM-02 FPGA module outputs data over USB 3.0
- **Host Computer**: VIA EPIA-P910-10Q Pico ITX board with on-board SSD for data storage
- **Microcontroller**: Collects auxiliary data from internal thermistors and CAN data packets from autopilot, interleaves all data for logging

**Key Specifications:**
- Sample rate: 60 MS/s
- ADC resolution: 14-bit
- Interface: USB 3.0 for data transfer
- Storage: Solid-state drive (SSD) with timestamp capability
- Power/Size: Meets sUAS constraints for weight and power consumption

## Products & Capabilities Described

**High Data Rate Compact Data Logger**
- High-speed data acquisition system designed for sUAS integration
- Captures data from multiple sources simultaneously (L-band antenna outputs, thermistors, autopilot state data)
- Includes onboard FPGA for digital signal processing (DSP) algorithm implementation
- CAN bus interface for integration with SwiftPilot autopilot system
- Enables future signal processing capabilities (e.g., FFT analysis)

**SwiftPilot Autopilot Integration**
- Optional CAN data packet integration providing auxiliary state data (position, altitude, attitude)
- Data used for geo-location of sensor measurements
- Enables precision flight control for low-altitude missions

## Use Cases & Applications

**Primary Application: Soil Moisture Mapping**
- Small UAS-based passive L-band microwave radiometer system
- Maps volumetric soil moisture content over agricultural plots
- Achieves 15-meter spatial resolution with 5-20 cm depth sensitivity
- Low-cost, simple-to-operate platform

**Supporting Measurements:**
- L-band antenna outputs (2 channels at high rate)
- Temperature monitoring via 8 thermistors (2 on antenna, 6 in LDCR)
- Radio frequency interference (RFI) mitigation algorithm testing

**Broader UAS Market Applications:**
- Emerging civil UAS market projected at $2.4 billion annual sales by 2020 (per AUVSI)
- System enables advanced sensing systems in sUAS through compact high-rate data logging
- Potential for follow-on sensing applications beyond radiometry
