## Executive Summary
Black Swift Technologies is a US-based Group 1 UAS manufacturer founded in 2011 that designs and manufactures purpose-built unmanned aircraft systems, flight management systems, and payloads for atmospheric research and extreme-environment operations. BST is distinguished as the only US domestic Group 1 UAS manufacturer with in-house aircraft design and flight management system capabilities, serving customers including USAF, NOAA, NASA, USGS, and leading research institutions.

## Company Overview
- Based in Boulder, CO; founded 2011
- Vertically integrated: designs and manufactures aircraft, avionics, flight control systems, payload interfaces, payloads, user interfaces, and ground stations
- All systems designed, manufactured, and serviced entirely in the USA
- Mission focus: specialized atmospheric research in extreme conditions (volcanoes, tornadoes, hurricanes, wildland fires)

## Core Competencies
1. **Aircraft Design and Control** - in-house design capability
2. **UAS Operations in Extreme Environments**
3. **Payload Integration and Deployment** - multispectral/RGB cameras, LiDAR, EO/IR, trace gas sensors, nephelometer, temperature/pressure/humidity/wind sensors, L-band radiometer
4. **On-Board Augmented Intelligence** - avionics subsystem with ML algorithms for intelligent fault detection and predictive failure detection
5. **Unified Flight Management System** - end-to-end avionics for autonomous flight control and command
6. **AI, Machine Learning, and Machine Vision Software Development**

## Key Differentiators
- Only U.S. Group 1 UAS manufacturer with proprietary flight management system and aircraft design capability
- **Data-Centric Flight Control** - payload-focused control system with real-time telemetry via autopilot data link for autonomous flight
- **Modular Field-Swappable Payload System** - rapid payload changes in field using standardized power, data, mechanical interfaces; no specialized tools required
- **SwiftCore FMS "Smart" Control** - sensor-based autonomous flight path modification to minimize operator workload and improve data quality
- All code and design created in-house for security and optimization (vs. open-source alternatives)
- Sandboxed mission-critical core with open interfaces for third-party integration

## Products & Capabilities Described

### Black Swift S2 UAS (Fixed-Wing)
- **What it is:** Rugged fixed-wing workhorse with modular, field-swappable payload nose cone
- **Key specifications:**
  - Payload capacity: 5 pounds (larger than competitive aircraft in class)
  - Endurance: 110 minutes
  - Ceiling: 6,000 meters (20,000 feet AGL)
  - Range: 110 km
- **Use cases:** Atmospheric research, volcano research, aerial mapping
- **Proven performance:** Arctic to Tropics operations

### Black Swift S0 UAS (VTOL/Tube-Launched Fixed-Wing)
- **What it is:** Small, ruggedized VTOL or tube-launched fixed-wing UAS
- **Payload capacity:** Full atmospheric sampling suite or small lightweight EO/IR cameras
- **Advantages:** Low-cost, economical to construct/operate/maintain; enables swarm solutions
- **Origin:** NOAA grant for air deployment from P-3 Orion "Hurricane Hunter" aircraft
- **Purpose:** Autonomous boundary layer measurements in hurricane eyewall

### Black Swift E2 Multirotor UAS
- **What it is:** Multi-rotor system originally developed for infrastructure inspection
- **Key specifications:**
  - Flight duration: 35 minutes
  - Payload: 6.5 pounds
  - Field-swappable payload capability
- **Rugged design:** Built for demanding field conditions, inclement weather operation

### SwiftCore™ Flight Management System (FMS)
- **What it is:** Fully integrated autopilot, tablet-based ground control station, intuitive GUI
- **Architecture:** Mission-critical sandboxed core with open interfaces for third-party integration
- **Differentiation:** All code and design created in-house; fully vetted and secured (vs. open-source alternatives)
- **Capability:** Sensor-based autonomous control enabling real-time telemetry and fully autonomous flight
- **User interface:** Tablet-based ground control station

### SwiftFlow 3D Wind Probe
- **What it is:** Integrated meteorological sensor
- **Application:** Boundary layer condition studies in hurricane eyewall
- **Delivery method:** Integrated into S0 for air deployment from P-3 Orion

## Use Cases & Applications

### Current/Active Projects
1. **AFWERX 20.3 Phase I SBIR** - AI/ML predictive and preventive maintenance algorithms for UAS aircraft
2. **NOAA Phase II SBIR** - Small, low-cost UAS air-dropped from P-3 Orion for hurricane boundary layer studies with integrated SwiftFlow 3D Wind Probe
3. **NASA Phase II SBIR** - UAS for upper atmospheric observations of Venus; survives harsh wind environment, performs atmospheric sampling, extracts energy on dark side
4. **NOAA Phase II SBIR** - AI-enabled GPS-denied navigation for BVLOS coastal UAS mapping missions
5. **Alerion Commercial Project (Spanish Partner)** - Multirotor system with advanced AI navigation for autonomous wind turbine inspection using LiDAR and RGB cameras in extreme conditions
6. **Major Cell Phone Provider (Commercial)** - UAS monitoring and mapping of 5G signal strength
7. **Air Force Phase II SBIR** - Small, low-cost, high-performance VTOL fixed-wing UAS for autonomous atmospheric profiling supporting parachute drops and localized weather forecasting (Air Force, Army, Special Operations Forces)

### Past Achievements (Completed SBIR Phases)
1. **NASA Phase II SBIR** - Machine vision automated safe landing system for distressed UAS (increase NAS safety and viability)
2. **NASA Phase II SBIR** - AI-enabled predictive/preventive maintenance system for UAS; ML algorithms provide early warning and diagnostics of critical system failures on USAF drones
3. **NASA Phase II SBIR** - Soil moisture management system using L-band radiometer integrated into S2 with multispectral post-processing software for unprecedented detail and resolution
