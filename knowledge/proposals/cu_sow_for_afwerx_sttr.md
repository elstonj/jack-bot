# CU SoW for AFWERX STTR

## Document Metadata
- Type: Statement of Work (subcontract)
- Client/Agency: NASA (via AFWERX)
- Program/Solicitation: SBIR Phase 1 (AF_X24E STTR)
- Date: March 3, 2025
- BST Products/Systems Referenced: Uncrewed aircraft system (UAS)
- Key Personnel: Dr. Christoffer Heckman (University of Colorado Boulder, Associate Professor, Department of Computer Science)

## Executive Summary
Black Swift Technologies is partnering with Dr. Christoffer Heckman at the University of Colorado Boulder to develop technical approaches and algorithms for GPS-denied and terrain-aware navigation capabilities under a NASA SBIR Phase 1 award. Dr. Heckman will provide expertise in algorithm selection, sensor integration, and will assist with Phase 2 proposal development.

## Technical Approach
- Development of technical approach and algorithms for terrain-aware navigation and GPS-denied capability using the same sensor suite and hardware
- Algorithm selection and customization based on Dr. Heckman's expertise
- Guidance on sensor selection, algorithm selection, and integration with existing uncrewed aircraft system
- Development of implementation plan for Phase 2

## Products & Capabilities Described

### Uncrewed Aircraft System (UAS)
- BST's existing uncrewed aircraft platform that will be enhanced with new navigation capabilities
- Will integrate new sensors and algorithms for GPS-denied operations

### Capabilities Under Development
1. **GPS-Denied Visual Navigation** - autonomous navigation without GPS using onboard sensors
2. **Terrain Relative Flight** - flight operations relative to terrain features
3. **Obstacle Avoidance** - autonomous detection and avoidance of obstacles

## Use Cases & Applications
Not explicitly detailed in this SoW; focus is on algorithm and technical approach development. Anticipated applications are GPS-denied environments and terrain-challenging operational areas.

- Period of Performance: 180 days
