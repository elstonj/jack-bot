## Executive Summary
Black Swift Technologies is developing modifications to its L-band soil moisture sensing system and E2 small UAS to enable runway integrity validation for the U.S. Air Force through real-time soil moisture and cone index measurements. The project involves upgrading the radiometer with improved RFI mitigation and calibration capabilities, integrating auxiliary thermal and NDVI sensors, and developing algorithms to correlate soil moisture with soil integrity metrics for operational use at Air Force bases.

## Technical Approach

**Radiometer Modifications (Milestone 2 - Completed):**
- Designed new revision of L-band Dual-Channel Correlator Radiometer (LDCR) with:
  - Real-time in-flight calibration using interior references with adjustable, wide temperature range
  - Enhanced RFI (radio frequency interference) mitigation
  - Digital correlator for improved RFI detection
  - Better immunity to gain fluctuation and thermal drift
- RF chain design includes integrated 90° hybrid coupler to mix upwelling and downwelling antenna temperatures
- Three amplifier stages optimally driving down converter/digital back end
- Lumped element 0°/180° hybrid at L-band: gain imbalance <0.15 dB, phase imbalance <0.5°
- 2x2 microstrip patch antenna array (vs. previous MiCo antenna) providing ~40° beamwidth (~0.8x flight height antenna footprint) and ~35 MHz bandwidth

**UAS Modifications (Milestone 3 - Mostly Completed Ahead of Schedule):**
- Integration of two auxiliary sensor payloads onto E2 platform:
  - NDVI sensor (two-band: 650nm, 810nm)
  - Thermal/infrared sensor (-70°C to 380°C range)
- Embedded data processing via Raspberry Pi Zero running C++ code
- Local data logging over USB, transmission to E2 via UART
- Self-contained bolt-on sensor module design for E2

**Algorithm Development (Milestone 4 - Planned):**
- Developing algorithms to compute soil integrity (cone index) as function of L-band soil moisture data and soil type

## Products & Capabilities Described

### E2 UAS Platform
- Small unmanned aerial system serving as primary platform
- Equipped with payload power and UART data interface
- Hosts radiometer, thermal sensor, and NDVI sensor payloads
- Flight-tested at 60m altitude for sensor validation

### L-Band Dual-Channel Correlator Radiometer (LDCR)
- Passive microwave radiometer for soil moisture measurement
- Updated version with improved RFI mitigation and digital correlator
- In-flight calibration capability
- 2x2 patch antenna array with 35 MHz bandwidth, ~40° beamwidth
- Integrated into E2 for airborne soil moisture mapping

### NDVI Sensor Package
- **Specifications:**
  - Two-band reflectance (650nm, 810nm)
  - Upward/downward-facing measurements for NDVI calculation
  - Range: 2x full sunlight
  - Calibration uncertainty: ±5%
  - Temperature response: <0.1%/°C
  - Measurement repeatability: <1%
  - Data rate: every 2 seconds (SDI-12 protocol)

### Thermal Sensor
- **Specifications:**
  - Ambient-compensated infrared
  - Range: -70°C to 380°C
  - Resolution: 0.02°C
  - FOV: 35°
  - Accuracy: ±0.3°C (22-40°C range), ±0.5°C (0-60°C range)
  - Data rate: near-instantaneous (I2C protocol)

## Use Cases & Applications

**Primary Use Case: Runway Integrity Assessment**
- Validate runway structural integrity via soil moisture and cone index measurements
- Enable Air Force rapid assessment of runway condition for operational readiness
- Support at Tyndall Air Force Base and other customer-defined locations

**Secondary Applications:**
- Wildfire response: NASA/USGS partnership proposal submitted for rapid soil moisture mapping using swarm of 8 soil moisture UAS
- NISAR satellite integration for rapid area mapping based on satellite-derived data

## Key Results

**Milestone 2 (Sensing System Modifications):**
- Radiometer redesign complete with significant performance improvements
- EMI (electromagnetic interference) mitigation plan finalized
- Hardware design complete; lab validation awaiting hardware delivery
- All design work for Milestone 2 completed

**Milestone 3 (UAS Modifications) - Ahead of Schedule:**
- Thermal and NDVI sensor selection and procurement complete
- Interface design and data logger complete
- 3D CAD and UAS integration design complete
- Preliminary build and testing completed; additional testing ongoing

**Field Testing Results:**
- Flew at 60m altitude test area (sod farm with varied terrain/vegetation)
- NDVI sensor comparison with higher-resolution Altum camera: very good match with slight bias but acceptable for replacement capability
- Thermal sensor comparison: bias observed (possibly ±5°C camera error); additional ground truth testing needed
- Demonstrated that compact, integrated NDVI sensor is viable replacement for larger Altum system despite post-processing requirement conflicts with AF timelines
