# BST Statement of Work: NSF CO/WY Climate Engine Wildfire Project

## Document Metadata
- Type: Statement of Work (SOW)
- Client/Agency: Cooperative Institute for Research in the Atmosphere (CIRA), Colorado State University
- Program/Solicitation: NSF CO/WY Climate Engine - Wildfire monitoring project
- Date: August 26, 2024
- BST Products/Systems Referenced: S2 (fixed-wing UAS), Soil Moisture Sensor, NOAA NightFOX payload
- Key Personnel: Beck Cotter (last editor), Dr. Kyle Hilburn (CIRA recipient)

## Executive Summary
BST will partner with CIRA at Colorado State University throughout 2025 to support wildfire monitoring and prediction efforts. The work involves integrating BST and NOAA sensor data into WRF-SFIRE modeling, collecting fuel loading data, and conducting rapid S2 deployments equipped with NOAA NightFOX thermal detection payload.

## Technical Approach

### Data Integration & Modeling
- Convert NOAA NightFOX hotspot detections and BST Soil Moisture Sensor data into formats compatible with WRF-SFIRE (Weather Research and Forecasting model coupled with fire spread simulation)
- Enable model ingestion of real-time thermal and soil moisture observations

### Field Operations
- Collect Soil Moisture and NDVI (Normalized Difference Vegetation Index) data over areas of interest to assess fuel loading conditions
- Deploy S2 UAS rapidly in response to satellite-based lightning detections, guided by CIRA operational requirements

## Products & Capabilities Described

### S2 UAS
- BST's fixed-wing unmanned aircraft system
- To be equipped with NOAA NightFOX thermal imaging payload for wildfire hotspot detection
- Capable of rapid deployment based on real-time satellite data triggers

### Soil Moisture Sensor
- BST proprietary sensor system
- Provides soil moisture measurements for fuel loading assessment
- Data to be integrated into WRF-SFIRE model

### NOAA NightFOX Payload
- Thermal hotspot detection system (NOAA-provided)
- To be integrated on S2 platform
- Generates data for fire detection and prediction modeling

## Use Cases & Applications
- **Wildfire Fuel Loading Assessment**: Using soil moisture and NDVI data to characterize vegetation conditions and fire risk
- **Real-Time Fire Detection & Monitoring**: Rapid S2 deployment following lightning detection events
- **Fire Spread Prediction**: Integration of sensor data into WRF-SFIRE model for improved fire behavior forecasting
- **Colorado/Wyoming Regional Focus**: Project covers NSF Climate Engine region in mountain/high-elevation wildfire zones

## Key Deliverables
1. Data format conversion specifications for NightFOX and soil moisture data
2. Soil moisture and NDVI dataset(s) from area(s) of interest
3. Rapid deployment operations (number unspecified in SOW)
4. Meeting support, reports, and project coordination activities
