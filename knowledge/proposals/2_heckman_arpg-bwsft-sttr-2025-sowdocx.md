# NASA STTR Phase I Statement of Work - GPS-Denied Navigation

## Document Metadata
- Type: Statement of Work (SOW)
- Client/Agency: NASA
- Program/Solicitation: NASA STTR (Small Business Technology Transfer) Phase I
- Date: March 7, 2025
- BST Products/Systems Referenced: Uncrewed aircraft system (not specified by model)
- Key Personnel: Dr. Christoffer Heckman (University of Colorado Boulder, Associate Professor); Beck Cotter (last editor, BST)

## Executive Summary
Black Swift Technologies is partnering with Dr. Christoffer Heckman at the University of Colorado Boulder under a NASA STTR Phase I award to develop GPS-denied navigation and terrain awareness capabilities for uncrewed aircraft. The work focuses on developing technical approaches and algorithms that operate with a single sensor suite and hardware platform.

## Technical Approach
- Develop technical approach and algorithms for three primary capabilities using a unified sensor suite and hardware
- Dr. Heckman will provide expertise in:
  - Sensor selection
  - Algorithm selection and customization
  - Integration with existing uncrewed aircraft system
  - Plan of work for implementation
- Focus on transitioning existing technologies to implement the target capabilities

## Products & Capabilities Described
**Uncrewed Aircraft System**
- Not specified by model/designation in this SOW
- Will be integrated with new navigation and terrain awareness capabilities

**Capabilities Under Development:**
1. GPS-Denied Visual Navigation
2. Terrain Relative Flight
3. Obstacle Avoidance

All three capabilities to be implemented with the same sensor suite and hardware.

## Use Cases & Applications
GPS-denied operations for uncrewed aircraft; terrain-relative navigation; autonomous obstacle avoidance in GNSS-denied environments.

## Notable Details
- This is a formal partnership between BST (prime contractor) and University of Colorado Boulder (subcontractor)
- Dr. Heckman will also assist with Phase 2 proposal development, including technical sections and plan of work
- Emphasis on integrating multiple capabilities into a single sensor/hardware platform for cost and complexity efficiency
- Phase 2 continuation anticipated
