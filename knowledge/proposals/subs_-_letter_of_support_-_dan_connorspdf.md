# Letter of Support - Dan Connors for NASA SBIR Phase II

## Document Metadata
- Type: Letter of Support (subcontractor commitment)
- Client/Agency: NASA
- Program/Solicitation: NASA SBIR Phase II - "Improved UAS Robustness through Augmented Onboard Intelligence"
- Date: December 5, 2017
- BST Products/Systems Referenced: S2
- Key Personnel: Dr. Jack Elston (BST), Dan Connors (subcontractor)

## Executive Summary
Dan Connors, PhD, commits to serve as a subcontractor on Black Swift Technologies' NASA SBIR Phase II proposal to develop improved UAS robustness through augmented onboard intelligence. He will contribute expertise in image processing, computer vision, and embedded machine learning, focusing on autonomous fault detection and safe landing zone identification algorithms.

## Technical Approach
Connors proposed to combine computer vision and machine learning algorithms on image and fault datasets to automate feature classification using on-board computing platforms. Specific work includes:
- Development of machine learning framework for identification of faults and required maintenance
- Development of framework for safe landing zone determination through visual cues
- Implementation of safe landing zone identification simulation using a priori wind information and photos captured during previous BST flights
- Implementation of safe landing zone identification using Nvidia Tegra hardware for integration onboard S2
- Phase II flight testing and validation
- Analysis of flight data and algorithm performance post-testing

## Products & Capabilities Described
**S2 UAS:**
- Target platform for integration of Nvidia Tegra-based safe landing zone identification system
- Will undergo flight testing of developed algorithms

**Nvidia Tegra:**
- Onboard computing hardware to be used for real-time safe landing zone identification implementation

## Use Cases & Applications
- **Autonomous fault detection:** Machine learning identification of maintenance needs
- **Safe landing zone identification:** Visual recognition and autonomous selection of safe landing areas
- **UAS integration into National Airspace:** Autonomous capabilities to support airspace integration
- Applications in Earth science, academic research, and commercial sectors
