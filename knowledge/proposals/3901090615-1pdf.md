## Executive Summary
The MLX90615 is a factory-calibrated, non-contact infrared thermometer IC integrating an IR thermopile detector and signal conditioning electronics in a single TO-46 package. It provides high-accuracy temperature measurements (±0.5°C over 0-50°C range) with 0.02°C resolution via either SMBus 2-wire interface or PWM output, making it suitable for compact, low-power thermal sensing applications including medical, industrial, and remote monitoring.

## Technical Approach & Architecture

**Core Design:**
- Integrated IR thermopile detector + MLX90325 signal conditioning chip in TO-46 can package
- Low-noise amplifier with programmable gain (1x, 10x, 40x)
- 16-bit Sigma-Delta ADC
- Powerful on-chip DSP for signal processing
- Embedded 65536-tap FIR low-pass filter
- Programmable IIR filter in EEPROM for noise/settling time trade-off

**Measurement Principle:**
- Thermopile converts IR radiation to voltage
- Amplified signal converted to 1-bit stream by Sigma-Delta modulator
- DSP processes to calculate both object temperature (TO) and ambient/package temperature (TA)
- Integrated PTAT (Proportional To Absolute Temperature) sensor for die temperature measurement

## Products & Capabilities Described

### MLX90615 Infrared Thermometer

**Specifications:**
- **Temperature Range (Object):** -40°C to +115°C
- **Temperature Range (Ambient/Sensor Die):** -20°C to +85°C
- **Accuracy:** ±0.5°C (0-50°C for both TA and TO); ±1.5°C to ±3°C across full range
- **Resolution:** 0.02°C (both measurement and output)
- **Supply Voltage:** 2.6-3.4V (3V nominal)
- **Supply Current:** 1.3-1.5 mA (normal operation); 1.1-3 µA (sleep mode)
- **Package:** Industry standard TO-46 can
- **Field of View (FOV):** Two options - 100°±5° (AAA variant) or 80°±5° (GAG variant)

**Optical Filter:** 5.5µm-14µm long-wave pass filter integrated; blocks visible and near-IR radiation

**Communication Interfaces:**
1. **SMBus (2-wire, factory default):**
   - SMBus-compatible protocol with 7-bit slave addressing (supports up to 127 devices)
   - Default slave address: 0x5B
   - Clock frequency: 10-100 kHz
   - Supports Read Word and Write Word commands
   - 8-bit Packet Error Code (PEC) with CRC-8 polynomial
   - Sleep mode support (1.1-3 µA consumption)
   - Wake-up from sleep via SCL low pulse >8ms

2. **PWM (10-bit output):**
   - Two frequency options: 1 kHz (high, 1.024ms period) or 10 Hz (low, 102.4ms period)
   - Customizable temperature range via EEPROM settings
   - Single-wire operation with weak pull-up (300kΩ typical)
   - Open-drain NMOS output
   - Maximum duty cycle limited to 62.5% (ensures line remains dynamic)

**Memory:**
- **EEPROM (16×16):** 256 bits total
  - Factory calibration data at addresses 0x04-0x0D (must not be altered)
  - Customer-writable addresses: 0x00-0x03 (SMBus slave address, PWM settings, config, emissivity)
  - Erase/write cycles: 100,000 @ 25°C; 40,000 @ 85°C
  - Data retention: 10 years @ 85°C
  - Erase/write time: 5ms each

- **RAM (16×16):** Runtime registers
  - Raw IR data (address 0x05)
  - TA - ambient/die temperature (0x06)
  - TO - object temperature (0x07)

**Power Management:**
- Power-On-Reset (POR) circuit; threshold 0.8-1.9V; valid data 0.5µs after POR
- Sleep mode: SCL high, SDA high (recommended); 1.1-3 µA drain
- Battery-compatible operation

**Calibration & Accuracy:**
- Factory calibrated across wide range with medical-grade accuracy
- Embedded emissivity compensation (adjustable 0.1-1.0 without recalibration)
- High (medical) accuracy calibration option available
- Internal oscillator factory-calibrated; PWM period stability ±15%

**Conversion Formulas:**
- Object Temperature: TO = (T_REG × 0.02) - K [degrees Celsius]
- Ambient Temperature: TA = (T_REG × 0.02) - K [degrees Celsius]
- PWM Output: Tout = (t/T - 0.125) × (TMAX - TMIN) + TMIN

## Notable Details

**Design Advantages:**
- Integrated solution (detector + signal conditioning in single package)
- Dual communication modes without hardware changes (software/EEPROM config)
- Low cost + small size + low power consumption ideal for portable/handheld applications
- Supports networked multi-sensor deployments (127 devices on 2-wire SMBus)
- Factory calibration eliminates user recalibration requirement
- Programmable IIR filter allows noise-vs-settling-time trade-off

**Critical Application Notes:**
- IR measurements inherently susceptible to thermal gradients (sensor care required)
- Emissivity compensation essential for low-emissivity materials (aluminum 0.18 vs. plastic 0.84-0.95)
- Field-of-view obstruction causes measurement errors (requires proper window/opening)
- Contamination on optical input degrades accuracy
- Environmental IR reflection from low-emissivity objects introduces errors
- Sunlight through windows has minor impact (glass IR-opaque, visible light filtered)
- Slow power-up or high-temperature stress (>absolute max) may cause loss of hermeticity

**Manufacturing/Reliability:**
- RoHS compliant
- ESD sensitivity: 2kV (AEC Q100 002)
- Qualified for reflow soldering, wave soldering, iron soldering per IPC/JEDEC standards
- Operating temp: -20 to +85°C; Storage: -20 to +125°C
- Maximum supply voltage: 5V (absolute); 3.6V (operating)

**Bus/Protocol Robustness:**
- Schmidt triggers on SMBus inputs (less sensitive to rise time)
- Can operate with higher pull-up currents for capacitive loading mitigation
- PWM output more robust to EMI than SMBus for remote installations
- Supports both low-power and high-power SMBus configurations

**Customization Options:**
- Temperature range for PWM: adjustable via EEPROM (0.02°C resolution)
- Emissivity: 0.1-1.0 selectable per object type
- IIR filter: 7 selectable response curves (IIR 2-8)
- Amplifier gain: 4 settings (1x, 10x, 40x, 40x)
- FOV: 100° or 80° package options
