## Product Description: DITEX Telemetry Servos

### Key Technical Features
- **Digital architecture**: Replace analog potentiometers and inefficient motor drives with digital position encoders and 32-bit processor
- **Premium coreless motors**: High torque-to-weight ratio with efficiency mapping and power metering to prevent heat damage
- **Programmability**: USB adapter with DITEX Manager Windows software for parameter configuration and behavior customization
- **Telemetry integration**: Compatible with JETI DUPLEX telemetry system; servo data delivered via signal wire
