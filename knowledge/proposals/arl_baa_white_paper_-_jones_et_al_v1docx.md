## Executive Summary
Black Swift Technologies proposed a three-year collaborative effort (2017–2019) to develop Army-relevant capabilities in data assimilation and UAS-based remote sensing for soil moisture characterization and boundary layer meteorology. The work integrates soil moisture data assimilation, UAS remote sensing validation, and numerical weather prediction (NWP) error characterization to support target recognition, trafficability assessment, and artillery meteorology applications.

## Technical Approach

### Theme A: Boundary Layer Data Assimilation at Army-relevant Scales
- Develop soil moisture data assimilation and surface energy balance modeling for boundary layer improvement
- Conduct innovation vector analysis on soil moisture estimates using existing downscaled datasets
- Design and implement Army-scale data assimilation components and synthetic experiments
- Apply innovation vector analysis to Reynolds Creek, ID and White Sands, NM field datasets in collaboration with ARL/BED
- Focus on data distribution-dependent design for data assimilation systems

### Theme B: UAS Remote Sensing of Soil Moisture and Land Surface Characteristics
- Deploy UAS-based remote sensing platforms for fine-scale soil moisture measurement
- Year 1 (2017): Prepare for Reynolds Creek field experiments; coordinate with team on microwave sensor forward operator simulation capabilities
- Year 2 (2018): Conduct Reynolds Creek, ID field experiment; perform calibration/validation intercomparisons; characterize sensor and model initialization errors
- Year 3 (2019): Conduct White Sands, NM field experiment; integrate characterized microwave UAS sensor data into Army-scale data assimilation systems

### Theme C: UKMET Unified Model NWP Error Characterization
- Perform error characterization of the UKMET Unified Model for Artillery Met and Boundary Layer applications
- Analyze model biases and error distribution patterns
- Explore atmospheric profiling capabilities; leverage balloon sounding data from ARL collaborators

## Products & Capabilities Described

**UAS Remote Sensing System**
- Purpose: Fine-scale soil moisture measurement and land surface characterization
- Equipped with microwave sensors for soil moisture estimation
- Used for calibration/validation of data assimilation algorithms
- Supports error characterization of sensor and model initialization

**Data Assimilation Framework**
- Army-scale soil moisture data assimilation system
- Integration of innovation vector analysis methodology
- Support for data distribution-dependent design
- Synthetic experiment capabilities for validation

**Microwave Sensor Forward Operator**
- Simulation tools for microwave sensor behavior
- Support for cal/val activities and error characterization

## Use Cases & Applications

- **Artillery Meteorology (Artillery Met)**: Improved meteorological support for artillery targeting and fire control
- **Boundary Layer Characterization**: Enhanced understanding of boundary layer phenomena for military operations
- **Target Recognition**: Soil moisture and surface characteristics as inputs for target detection and classification
- **Trafficability Assessment**: Soil moisture data to support military vehicle mobility predictions
- **Army-scale Operations**: Capabilities scaled appropriately for Army operational planning and tactical support

## Field Sites and Collaborations

- **Reynolds Creek Experimental Watershed, Idaho**: Primary field validation site for soil moisture and boundary layer measurements (experiments planned for summer 2018)
- **White Sands Missile Range, New Mexico**: Secondary field site for extended validation; collaboration with ARL/Battlefield Environment Division (ARL/BED)
- **Academic Partners**: Colorado State University (Andrew S. Jones, Steven J. Fletcher, Jeffrey D. Niemann) and University of Colorado–Boulder (Al Gasiewski)

## Requested Resources

- Boundary layer modeling coordination with ARL/BED
- Access to Army High-Performance Computing (HPC) facilities for NWP data analysis
