## Executive Summary

**Note:** This document is from HF Designworks, Inc., not Black Swift Technologies. HF Designworks is a human factors, user interface design, and software development company founded in 2006 that specializes in making device interfaces more usable, effective, and safer. The document presents their capabilities across multiple domains including medical software, command & control systems, robotic control, web and mobile development.

## Products & Applications Described

### 1. **Gaming Network Cards**
- Complete iterative UI overhaul with user feedback
- Usability testing of existing and proposed designs
- Industry-leading UI with real-time, dynamic status information
- Only high-performance network card built specifically for online gaming

### 2. **Medical Software**
- Physician and Nurse digital patient information annotation and exchange
- Electronic Medical Records (EMR) systems
- Cross-platform consistent interfaces
- Implementation of non-traditional information features (e.g., "sparklines")
- Documented design patterns

### 3. **Command & Control (C2) Systems**
- Force XXI Battle Command Brigade and Below (FBCB2) – digital mapping and communications
- Future Force Warrior (FFW) – C2 for foot soldiers
- iWarrior Commanders Visualization Tool
- Scaled, touch-enabled interfaces for command and control of soldiers and unmanned systems
- Mapping conversion applications: GPS track data to density heat maps to KML images
- Compatibility with Google Earth viewing across Windows, Web, and Android applications

### 4. **Robotic Control (UGV/UAV)**
- Ground and Air Robotic Control interfaces
- Scalable platforms: Ground Control Stations to multi-screen vehicle-mounted to PDA
- Both autonomous and semi-autonomous platform consideration
- 1:1 and 1:N operator control configurations
- Soldier testing and feedback integration

### 5. **Social Networking (Soldier-Centric)**
- Web-based social network concepts for warfighters
- Secure, anonymous, supportive environment for sharing experiences and struggles

### 6. **Distributed Team Workload Analysis Tool**
- Defines and assesses distributed team workload
- Leverages subjective and objective data collection
- Developed for Infantry Mission Scenarios
- Scenarios under development: Command & Control, UAS, Medical

### 7. **Multi-Display System UI Design**
- UI design for varied usage environments (in-vehicle/out of vehicle, performance-oriented)
- Scaling designs from multi-display systems to small PDA-sized systems
- Extensive touch screen display UI design experience

### 8. **Web Development**
- Web designers, graphics designers, and user interface engineers
- Portfolio includes commercial websites and applications

### 9. **Mobile Development (Android & iOS)**
- Cross-platform mobile applications
