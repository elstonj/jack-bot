## Executive Summary
This proposal outlines an integrated methane detection and monitoring system for orphaned well sites, combining Black Swift Technologies' drone platforms with flux towers, continuous ground monitoring, and environmental analysis software. The project aims to advance FluxMapper™ technology from TRL-4/5 to TRL-6, achieving 0.2 kg/h methane detection sensitivity while demonstrating a cost-effective, scalable alternative to traditional orphaned well monitoring approaches.

## Technical Approach
The project proposes a multi-platform, integrated monitoring solution:

- **Drone-based monitoring**: Black Swift S2 UAS equipped with Quanta3 methane sensor for aerial detection and mapping
- **Ground-based continuous monitoring**: Eddy covariance towers with LI-COR methane evaluation systems
- **Data analysis & integration**: FluxMapper™ software for processing and analyzing methane flux data
- **Remote connectivity**: HOO-nah device (HPWT) for reliable data transmission from remote sites
- **Three-phase development**: Structured performance periods with Go/No-Go decision points to de-risk advancement

## Products & Capabilities Described

- Continuous ground-based monitoring infrastructure
- Equipped with LI-COR methane sensors for atmospheric flux measurement

### HOO-nah Device (HPWT)
- Remote connectivity solution for data transmission from orphaned well sites
- Enables reliable remote connectivity in locations with limited infrastructure
