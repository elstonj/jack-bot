# A Scalable System for Supplemental Meteorological Information Using UAS-based Wind and Eddy Dissipation Measurement

## Document Metadata
- Type: SBIR Phase I Proposal Briefing Chart
- Client/Agency: NASA
- Program/Solicitation: 2022 SBIR, Topic A2.03-2738
- Date: March 9, 2022
- BST Products/Systems Referenced: S2 (fixed-wing UAS)
- Key Personnel: Jack Elston (PI, Black Swift Technologies)

## Executive Summary
Black Swift Technologies proposes developing WATCHES (Winds Aloft, Turbulence Characteristics and Hazardous Eddy Surveillance), a scalable module to measure winds and turbulence in the lowest 5,000 feet of atmosphere using small UAS. In partnership with NCAR, the system will adapt eddy dissipation rate (EDR) algorithms to fixed-wing UAS platforms to fill critical data gaps needed for safe Urban Air Mobility (AAM/UAM) and UAS Traffic Management (UTM) operations.

## Technical Approach
- **Algorithm Adaptation**: Leverage BST's historical flight data collection to adapt EDR (Eddy Dissipation Rate) algorithms from manned aircraft to small fixed-wing UAS; assess whether existing UAS flight data meets EDR algorithm requirements
- **System Requirements Definition**: Identify and document computational power, sensors, and communications interfaces necessary for accurate EDR measurements
- **Hardware Development & Testing**: Develop bench-top demonstration running on flight-ready hardware; assemble and test complete WATCHES setup on S2 platform for Phase II flight testing
- **Vertical Winds Integration**: Install vertical winds algorithm on Black Swift fixed-wing UAS
- **Validation & Commercial Viability**: Perform validation testing; explore commercial potential and product viability

## Products & Capabilities Described

### S2 (Black Swift Fixed-Wing UAS)
- Proposed platform for WATCHES integration and flight testing
- Will carry the WATCHES measurement module with EDR and vertical winds algorithms
- Historical flight data from S2 will be used to validate and refine algorithms

### WATCHES System
- Modular design intended to be added to any small UAS
- Measures winds and turbulence profiles in lower atmosphere
- Provides real-time data dissemination to AAM/UAS/UAM operators
- Core capability: eddy dissipation rate measurement for turbulence characterization

## Use Cases & Applications

### NASA/Government Applications
- **UAS Traffic Management (UTM)**: Enable accurate 3D wind and turbulence profiles for safe UAS integration into national airspace
- **Urban Air Mobility (AAM/UAM) Safety**: Provide real-time turbulence hazard data to operators
- **Meteorological Research**: Fill data void in lower atmosphere observations (0-5,000 ft); support advances in short-term Numerical Weather Prediction
- **Scalable Monitoring**: Scale measurement capability as AAM industry grows to enable increased atmospheric sampling at minimal cost

### Commercial Applications
- **Direct Operator Subscription**: Real-time turbulence data to AAM/UAS/UAM operators
- **Third-Party Services**: Distribution through commercial weather/data services
- **Scientific Data Mining**: Commercial scientific applications

## Phase I Objectives & Deliverables

**Objective 1**: Adapt EDR algorithm for small fixed-wing UAS using BST historical flight data

**Objective 2**: Identify and document system requirements (processing power, sensors, communications)

**Objective 3**: Bench-top demonstration of WATCHES on Phase II flight-ready hardware

**Key Tasks**:
1. Implement wind and turbulence algorithms
2. Develop Interface Control Document
3. Develop plan to extend WATCHES capability
4. Install vertical winds algorithm on BST fixed-wing UAS
5. Validation testing
6. Explore commercial potential and product viability
7. Prepare and submit reports

**Phase I Milestones**:
1. Demonstrate WATCHES using S2 historical data
2. Complete Interface Control Document
3. Finalize hardware and sensor requirements for fixed-wing UAS
4. Assemble and test flight-ready WATCHES setup for S2 (Phase II entry)
