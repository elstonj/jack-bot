## Executive Summary
This academic paper presents trajectory optimization methods for small unmanned aerial vehicles (UAVs) exploiting dynamic soaring techniques to extract energy from linear wind gradients. Using three non-linear optimal control problems with genetic algorithm optimization in MATLAB, the researchers developed admissible loitering trajectories that maximize specific energy, maximize specific energy rate, or minimize control effort, demonstrating the feasibility of sustained, fuel-efficient flight without thrust.

## Technical Approach
- **System Model**: 3-DOF point mass aircraft dynamics without thrust component, governed by lift, drag, and weight
- **Control Variables**: Bank angle (φ) and lift coefficient (CL)
- **Wind Field Model**: Linear wind gradient varying with altitude in three regions:
  - Below 100m: no wind (Ws = 0)
  - 100-115m: linear gradient (Ws = 1 s⁻¹, We = Ws·h)
  - Above 115m: constant wind of 15 m/s
- **Optimization Method**: Piecewise optimal control using genetic algorithm; total flight time divided into six equal intervals with constant control values per interval
- **Objective Functions**:
  1. Maximize specific energy: J = Te + ∫(gh + 0.5(Vt)²)dt
  2. Maximize specific energy rate: Minimize J using Ė = gh·ḣ + Vt·V̇t
  3. Minimize control effort: J = Te + ∫(0.5(φ² + CL²))dt
- **Terminal Conditions**: Cost function enforces return to initial state (loiter pattern closure)

## Key Results

### Test Vehicle Specifications
- Mass: 5.44 kg
- Wing area: 0.957 m²
- Wing aspect ratio: 19.54
- Maximum lift coefficient tested: 1.329
- Maximum bank angle: ±12°

### Maximum Specific Energy Problem
- Starting altitude 92m: 44.29 J/kg gained, 6.72m height gain, 7s cycle
- Starting altitude 110m: 225.11 J/kg gained, 10s cycle
- Starting altitude 120m: 329.72 J/kg gained, 11s cycle
- Optimal starting point: below 100m altitude (steeper wind gradient)

### Maximum Specific Energy Rate Problem
- Starting altitude 92m: 44.13 W/kg, 1.65 m/s velocity increase, 6.25m height gain, 8s cycle
- Starting altitude 110m: 195.13 W/kg, 6s cycle
- Starting altitude 120m: 61.27 W/kg, 6s cycle
- Maximum power gained: 8.025 kW during flight
- Result shows positive energy contribution from wind gradient offsetting drag losses

### Minimum Control Effort Problem
- Starting altitude 92m: function value 282.77, 7s cycle
- Starting altitude 110m: function value 389.66, 6s cycle
- Starting altitude 120m: function value 297.08, 5s cycle
- Maximum velocity achieved: 6.5 m/s during cycle
- Control profiles are piecewise linear and discontinuous

## Notable Details
- **Genetic Algorithm Parameters**: Population size 20-75; function tolerance 1e⁻6; generation limit 100; initial penalty 10; scattered crossover with rank fitness scaling
- **Convergence**: Optimizer converged in 45-100 iterations
- **Key Finding**: Vehicles starting from altitudes below 100m produce optimal results across all three performance metrics due to steeper wind gradient proximity
- **Loiter vs. Other Patterns**: Loiter patterns produced smaller maximum altitude gains than travel or other basic patterns
- **Terminal Condition Satisfaction**: Terminal errors closely met for all three cases
- **Historical Context**: Dynamic soaring theoretically proposed by Lord Rayleigh (1883); modern application to UAVs emerged late 1990s; this work extends Zhao's (2004) three-case optimization with novel piecewise control approach
- **Comparison to Biology**: Albatrosses naturally perform dynamic soaring; study aims to replicate this capability in engineered systems

## Limitations & Future Work
- Linear wind gradient assumption (acknowledged that non-linear profiles exist)
- Loiter pattern focus (could extend to traveling patterns)
- Flat, non-rotating Earth assumption (valid for localized, short-duration flight)
- No experimental validation presented (simulation only)
