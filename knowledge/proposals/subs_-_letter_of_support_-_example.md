## Document Summary
This is a **template letter of support** for a NASA SBIR Phase II proposal. It is not a completed letter but rather a blank form showing the structure and language expected from a subcontractor committing to support BST's proposed project. The document provides placeholders for a subcontractor to express commitment to the Phase II effort and detail their proposed contribution.

## Project Context
- **Proposal Title**: "Improved UAS Robustness through Augmented Onboard Intelligence"
- **Funding Source**: NASA SBIR Phase II
- **Primary Contact**: Dr. Jack Elston, Black Swift Technologies LLC, Boulder, CO

## Template Structure
The letter includes sections for:
1. Statement of support and interest in the proposal
2. Explanation of subcontractor's relevant experience
3. Description of anticipated applications (integration into National Airspace, Earth science, academic research, commercial sector)
4. Statement of Work description (to be filled in)
5. Budget breakdown (consulting hours, travel, other costs)
6. Subcontractor leadership assignment
7. DUNS number reference

## Notable Details
- This is an **example/template document**, not a signed letter from an actual subcontractor
- Intended to guide subcontractors on format and content for letters of support
- Demonstrates BST's focus on UAS autonomy and integration into the National Airspace System
- Suggests the Phase II work targets multiple sectors: government, academia, and commercial applications
