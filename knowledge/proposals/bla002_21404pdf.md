## Executive Summary
Bartington Instruments provided a quotation to Black Swift Technologies for a complete 3-axis magnetometer calibration and testing laboratory system. The equipment package totals £70,505 (including delivery/insurance) and is intended to support BST's Navy SBIR Phase II magnetometer development program.

## Technical Approach
BST is procuring a comprehensive Helmholtz coil-based magnetic field calibration and measurement system from Bartington Instruments to validate magnetometer performance during development. The system enables precise characterization of 3-axis magnetic field sensors.

## Products & Capabilities Described

### Bartington Helmholtz Coil System Components:
- **BHC2000-3-A Ferronato Helmholtz Coils (3-axis)**: Primary calibration apparatus for generating controlled magnetic fields (£30,045)
- **PA1 Power Amplifier**: Drives the three-axis Helmholtz coil system (£13,260)
- **CU1 Control Unit**: Manages Helmholtz coil operation (£3,765)
- **MAG-13MS1000 3-Axis Magnetic Field Sensor**: Measurement sensor with 1000 nT range capability (£2,405)
- **PCIe DAQ Card (USB Version)**: Data acquisition interface for computer integration (£4,325)
- **BHC2000 DUT Stand**: Device Under Test mounting/positioning stand (£4,460)
- Supporting infrastructure: cabling (£320), harness (£380), jig (£705), packing/palletization (£1,420)

## Use Cases & Applications
Calibration and validation testing of magnetometer sensors during Phase II Navy SBIR development. The Helmholtz coil system allows generation of known magnetic field strengths and orientations for sensor accuracy characterization and performance verification.
