## Executive Summary
Black Swift Technologies completed a security program questionnaire for AFRL as part of their AFWERX SBIR Phase II award on soil moisture mapping. The questionnaire documents BST's physical security, information security, cybersecurity protocols, and personnel access controls across all aspects of the project.

## Security Practices Described

### Physical Security
- All entries to physical facilities are secured
- Each exterior entry door is locked

### Information Security Infrastructure
- All electronic data stored on secure commercial cloud-based platforms
- Password protection required on all employee devices
- Additional passwords required to access supplemental software tools/suites
- Leverages proven existing security processes from commercial vendors

### Data Storage & Transmission
- Information stored using BST's hardware devices and cloud-based software tools
- Passwords and multi-layer authentication on all data access points
- Same security protocols apply to transmission/transportation of information

### Access Controls
- Access restricted to employees directly associated with the specific effort
- Personnel safeguards in place to control information access
- Internal restrictions on information security aligned with contract terms

### Data Handling Procedures
- Strict information security protocols for disposal and destruction of information
- Reproduction procedures comply with contract-specified terms and client requirements
- GFE/GFI (Government Furnished Equipment/Information) safeguarded per contract requirements and internal protocols

### Cybersecurity & Network Protection
- Information security protocols based on exhaustive cybersecurity and network protection framework
- Industry-best cybersecurity standards and procedures incorporated
- Secure commercial platforms leverage proven security processes
- Multi-factor authentication (passwords on devices + software access passwords)

### Operational Security & Insider Threat Mitigation
- OPSEC processes prevent adversary access and project compromise
- Internal software and accountability mechanisms to detect and determine source of potential security concerns
- Cloud-based platforms provide audit trails and monitoring capabilities
- Multi-layer password protection deters insider threats

### Incident Response
- Capability to investigate compromised information through internal tools and accountability mechanisms
- Ability to identify "who and when" of potential security breaches

## Compliance & Training
- Willing to provide AFRL S&T Protection training to all personnel with access annually

## Notable Details
- Reliance on proven commercial cloud security platforms rather than proprietary systems
- Multi-layered password authentication approach (device-level and application-level)
