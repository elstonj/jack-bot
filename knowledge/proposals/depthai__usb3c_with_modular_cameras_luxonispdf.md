## Products & Capabilities Described
**DepthAI: USB3C with Modular Cameras (Part Number: BW1098FFC)**
- **What it is:** A platform for embedded Spatial AI combining depth perception, object detection (neural inference), and object tracking
- **Hardware composition:**
  - 1x Color 12MP modular camera
  - 2x Synchronized global-shutter 720P modular cameras for disparity depth measurement
  - USB3C interface
- **Key features:**
  - Modular camera design for custom integration and custom stereo baselines
  - Python API for ease of use
  - Includes firmware and software suite
- **Price:** $399.00
- **APIs available:** Python and C++
