## Key Procedures Documented

### Part 1: Navy Deliverable Submission
1. **Preparation (1-2 weeks before due date)**
   - Verify deliverable specifications in Asana or contract
   - Apply CUI markings (minimum CUI/Dist B for vendor confidentiality)
   - Format: PDF, max 25MB, named using full contract number and CLIN/Item number
   - Example filename format: N0001412C9999A001ProgressReport1.pdf

2. **Obtain Access**
   - Obtain Upload Code from Navy deliverable website (https://www.navysbirprogram.com/navydeliverables/default.aspx)
   - Upload code for this contract: LXONII5161
   - Login with firm name "Black Swift Technologies"
   - Use Point of Contact email: proposals@blackswifttech.com

3. **Submit via Navy Website**
   - Navigate to Contract Deliverables section
   - Select correct contract (N68335-25-C-0270)
   - Confirm CUI marking compliance
   - Upload PDF and select correct deliverable type per CLIN
   - Save and confirm file correctness

### Part 2: Invoice Submission to PIEE Website
1. **Prerequisites**
   - Create invoice in Quickbooks first (for tracking only, not sent to Navy)
   - Obtain invoice number from Quickbooks

2. **WAWF Submission Process (https://piee.eb.mil/)**
   - Select WAWF button under "Payment"
   - Verify CAGE Code/DUNS
   - Enter contract number (may be selectable after first submission)
   - Select appropriate CLIN number
   - Select "Invoice and Receiving Report (Combo)" document type
   - Set Inspection: Destination and Acceptance: Destination
   - Enter effective date (usually in past, after deliverable approval)
   - Provide invoice number, invoice date, shipment number (fabricated), and shipment date

3. **Line Item Requirements**
   - May require Qty Shipped = 1
   - Error messages regarding "Final Invoice already exists" may require explanatory comments

### Quickbooks Invoice Creation
- Navigate to project in Projects section
- Add Invoice from "Add to Project" dropdown
- Fill in:
  - Product/Service: Custom Aerospace Engineering (for this contract)
  - Description: Use CLIN description from contract
  - Qty: 1
  - Rate: Dollar amount associated with CLIN
  - Class: Government
  - Uncheck Tax box
- Save (not Send)—for internal tracking only

## Notable Details
- Upload code is per contract, not per deliverable (one code needed for entire contract)
- Navy deliverable website provides general guidance but omits critical steps (e.g., obtaining upload code)
- WAWF system occasionally has inconsistent requirements (some reviewers prefer "Invoice" instead of "Combo"); if rejected, try alternative format
- Multiple deliverables must be submitted separately (cannot batch upload multiple PDFs)
- Shipment information (number and date) required even though contract involves no physical shipments; placeholder data acceptable
- Internal references indicate use of AFWerx system (https://afwerx.experience.crmforce.mil/) for contract management with credentials operations@bst.aero
- Project tracking uses internal naming convention [550-2] for Navy STTR program
