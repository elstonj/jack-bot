## Document Purpose
This is a blank flight report template designed for submitting flight test data to NASA's Airborne Science Program. It was created as part of planning documentation for a Phase I SBIR project related to soil moisture monitoring.

## Template Structure

### Required Report Fields
- **Aircraft Information**: Aircraft type and identification
- **Operating Details**: 
  - Operating site(s) and flight route (From/To)
  - Flight date
  - Flight number
  - Take-off and landing times (local/GMT)
  - Total flight time
- **Flight Request Information**: Flight Request # and Principal Investigator designation
- **Flight Purpose**: Checkboxes for Data collection, Ferry, Functional Check, or Other
- **Aircraft Status**: Up or Down status
- **Sensor Payload**: Payload description field
- **Comments Section**: Open notes area
