# Submission Checklist for DoD SBIR Phase II Proposal

## Document Metadata
- Type: Submission checklist / guidance document
- Client/Agency: U.S. Air Force (DAF)
- Program/Solicitation: DoD SBIR Phase II
- Date: Created 2021-05-31; Modified 2021-06-02
- BST Products/Systems Referenced: None
- Key Personnel: None named

## Executive Summary
This document is a comprehensive checklist and preparation guidance for Phase II SBIR proposals submitted to the Air Force. It outlines all required documentation, formatting standards, and submission components across six proposal volumes, emphasizing the importance of demonstrating Phase I technical progress, dual-use applications, and commercial potential.

## Document Content
This is a procedural/administrative document rather than a substantive proposal or technical report. It does not contain information about Black Swift Technologies' products, capabilities, or technical approaches.

### Key Requirements Outlined:

**Registration Requirements:**
- DUNS Number
- CAGE Code
- DoD SBIR/STTR ID

**Volume 1 (Proposal Coversheet):** CAGE, DUNS, abstract, certification, contact info

**Volume 2 (Technical Proposal):** Table of contents, glossary, technical summary, commercialization potential, Phase I results, Phase II objectives/key results, work plan, milestones, deliverables, key personnel

**Volume 3 (Cost Volume):** Labor, overhead, G&A, subcontractors, materials/equipment/travel, cost share, profit

**Volume 4:** Company Commercialization Report

**Volume 5 (Supporting Documents):** Customer memorandum, letters of support, resumes, certifications, foreign citizen disclosure, technical data assertions, military critical technical data agreement (DD Form 2345)

**Volume 6:** Fraud, Waste, and Abuse training

### Formatting Standards:
- Minimum 12-pitch or 11-point font
- Standard 8.5" × 11" paper
- One-inch margins
- Consecutive page numbering
- No proprietary information to be included

## Notable Details
- Air Force emphasizes dual-use applications and commercial exploitation potential
- Proposals must demonstrate sufficient Phase I technical progress to justify Phase II funding
- All proprietary information must be clearly marked per AF X20.3 SBIR guidelines
- Signed Customer Memorandum is mandatory
