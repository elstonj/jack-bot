## Executive Summary
This document is a detailed engineering schematic and parts list for a P-3 Filter LNA (Low Noise Amplifier) Box, a specialized RF signal conditioning unit designed for NOAA aircraft telemetry systems. The device filters and amplifies signals for airborne data collection systems.

## Technical Approach
The device is designed as an inline RF signal conditioning unit that:
- Accepts input from AVAPS Telemetry Preamp via Type N male connectors
- Routes signal through a cavity filter (LARK 3C404-T6-3A) for frequency selectivity
- Amplifies filtered signal using a Mini-Circuits ZX60-14LN-S+ Low Noise Amplifier
- Provides regulated +6 VDC power from AC/DC converter (Acopian 6EB40)
- Outputs conditioned RF signal to telemetry antenna
- Includes 1.5A breaker protection and multiple SMA/Type N connectors

**Power Specifications:**
- Input: 105-125 VAC, 47-420 Hz
- Output: 6 VDC @ 400 mA
