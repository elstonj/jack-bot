# NASA 2016 SBIR Phase III – Volcano Monitoring RFP

## Document Metadata
- Type: RFP (Request for Proposal) – Statement of Work invitation
- Client/Agency: NASA
- Program/Solicitation: NASA SBIR Phase III (2016 Volcano topic)
- Date: December 14, 2021
- Submission Deadline: December 21, 2021, 3:30 p.m. CDT
- Company: Black Swift Technologies, LLC
- Contact: Jack Elston
- Contracting Officer: Steven Brockway (Research Activities Branch)

## Executive Summary
NASA issued a Firm-Fixed Price Phase III contract opportunity to Black Swift Technologies to complete research, development, and demonstration work related to volcano monitoring (based on prior SBIR Phase I/II work). The RFP requires a detailed technical proposal addressing specific Statement of Work tasks, along with comprehensive pricing and resource planning.

## Proposal Requirements
The RFP specified that Black Swift's proposal must include:

1. **Pricing Section**
   - Detailed labor category costs corresponding to each SOW section and sub-section
   - Firm-fixed price structure

2. **Technical Section**
   - Detailed Phase III R/R&D (Research and Reporting & Development) plan
   - For each task element: description of what will be done, where, and how
   - Detailed methods to achieve each task/objective
   - Task descriptions and schedules
   - Resource allocations
   - Estimated task hours for key personnel
   - Planned accomplishments and project milestones
