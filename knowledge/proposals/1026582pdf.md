# CFD Analysis of the SBXC Glider Airframe

## Document Metadata
- Type: Master's thesis
- Client/Agency: Naval Postgraduate School (NPS)
- Program/Solicitation: TaLEUAS (Tactical Long-Endurance Unmanned Aerial System) project
- Date: June 2016
- BST Products/Systems Referenced: SBXC Glider (note: developed by Reaction Research & Engineering/RNR, not BST)

## Executive Summary
This thesis develops and validates a computational fluid dynamics (CFD) methodology for simulating the aerodynamic performance of the SBXC glider airframe using ANSYS CFX. The research establishes a reusable process for glider design analysis, comparing CFD predictions against actual flight test data, and provides recommendations for improving simulation accuracy at higher velocities.

## Technical Approach

**CFD Methodology:**
- Used ANSYS CFX software with finite element methods (FEM) to model fluid behavior around the SBXC airframe
- Created 3D CAD model in SolidWorks using two techniques:
  - "Picture sketch" method for fuselage and stabilizers
  - Airfoil coordinate generator software for wing (developed by Kevin D. Jones) to create 18 wing sections based on S2048 airfoil modified with 7.4% thickness increase
- Generated fluid domain: 6.52 × 4 × 4 meters (approximately 10× root chord length in all directions)
- Used domain symmetry to model only half the domain for computational efficiency
- Analyzed range of velocities from stall to shallow dive conditions
- Mesh statistics: 141,111,355 elements, 49,254,491 nodes, average element quality metric of 0.62371

**Meshing Strategy:**
- Size function: "Proximity and Curvature" to refine mesh near leading edges
- Inflation technique: "Last Aspect Ratio" with first layer height of 1 × 10⁻⁵ meters for boundary layer modeling
- Y+ values controlled for proper boundary layer resolution

**Solver Configuration:**
- Material: Air at 25°C (isothermal analysis)
- Turbulence model: Shear Stress Transport (SST) with Gamma-Theta transition model
- Pressure-based solver
- Reference pressure: 1 atmosphere

## Key Results

**Validation Against Real Flight Data:**

Comparison between CFD predictions and actual flight performance:
- Best L/D (Lift/Drag Ratio):
  - CFD analysis: ~4.1 at 15 m/s (near optimal glide speed)
  - Real flight data (Edwards): ~4.0 at 15.5 m/s
  - Real flight data (Ellias): Similar region identified
  
**Performance Data Points (Table 15 reference):**
- Sink rate measurements across velocity range: 10–18 m/s tested
- Sink polar comparison showed CFD results matched actual flight behavior pattern
- Minimum sink rate predictions within acceptable range for lower velocities (≤15 m/s)

**Mesh Quality Assessment:**
- Average element quality metric: 0.62371 (above 0.3 minimum threshold)
- 49.3 million nodes successfully resolved boundary layer phenomena
- Y+ values maintained in appropriate range for SST turbulence model

**Key Findings:**
1. CFD methodology produces results matching experimental sink polar data for velocities up to ~15 m/s
2. Accuracy reduces for velocities >15 m/s; recommendations made for improvement
3. Process successfully simulates boundary layer behavior and pressure/velocity distributions
4. Established reproducible workflow for future airframe designs

**Aerodynamic Insights:**
- Validated that 7.4% thickness increase during manufacturing did not destroy laminar flow qualities of original S2048 airfoil
- S2048 airfoil shown to maintain good low-Reynolds number performance characteristics across tested range
- Boundary layer behavior successfully captured through mesh inflation strategy

**Previous Performance Data Compared:**
- John Ellias (2011): GPS-based performance testing producing sink polar; flights over 6 weeks
- Dan Edwards (2013): Piccolo autopilot-based testing at 20 Hz data rate with airspeed hold capability; produced updated sink polar equation: Vvert = 0.0095V²ₕₒᵣᵢz - 0.3782Vₕₒᵣᵢz + 4.6072

**Technical Recommendations:**
- Improve Y+ value control for velocities >15 m/s
- Consider mesh refinement in separation regions for higher-speed analysis
- Model provides good baseline for optimization studies of wing modifications, winglet designs, and control surface effectiveness
