## Executive Summary
This SOW establishes terms for joint development between Black Swift Technologies and Quanta3 LLC to integrate the QM3000-VA2 methane sensor onto the BST S2 airborne platform. The work consists of two tasks: performance enhancements to an existing sensor unit (System #1) owned by University of Colorado, and design/fabrication of a second sensor unit (System #2) for BST field testing with a prospective anchor customer (PDC).

## Technical Approach
- **Task 1**: Iterative performance enhancements to QM3000-VA2 System #1 based on Alaska field test results; lab validation; integration and field testing support with CU's S2 platform (completion target: end of January 2023)
- **Task 2**: Design and fabrication of System #2 based on System #1 improvements; component procurement; fabrication, testing, and iteration to performance specifications; delivery to BST for integration testing and field trials with PDC (February/March 2023)
- Key constraint: FPGA processing unit procurement delays due to chip shortages; loaner unit available only until end of March; permanent replacement unit on order (April 2023 ship date)

## Products & Capabilities Described

### S2 Airborne Platform
- BST's airborne platform for sensor integration
- Capable of hosting the QM3000-VA2 methane sensor system
- Field testing location: BST test field
- Intended for PDC customer deployment in February/March 2023

### QM3000-VA2 Methane Sensor (Quanta3)
- Integrated methane detection system developed by Quanta3 LLC
- System #1: Already delivered to University of Colorado; undergoing performance enhancement
- System #2: New unit to be built for BST; target for PDC field deployment
- Integration: Mounted on S2 airborne platform

## Use Cases & Applications
- **Alaska Scientific Deployment**: CU's System #1 scheduled for March 2023 deployment in Alaska
- **Methane Leak Detection**: Airborne detection and characterization of methane emissions
- **Anchor Customer (PDC)**: Prospective field testing planned for February/March 2023

## Specifications (Target Performance)

| Parameter | Value |
|-----------|-------|
| Mass | Maximum 2.27 kg (<5 lbs), excluding S2 airframe components |
| Power Consumption | Maximum peak: 20W; Nominal: ~10W |
| Airborne Sensitivity | <30 ppb (1 sigma, includes atmospheric variability) |
| Ground-based Precision | Better than 1 ppbv (demonstrated in System #1) |
| Measurement Frequency | 1 Hz (higher frequency planned for future models) |
