## Executive Summary
Technical wiring diagram for integrating a Black Swift UAS with NOAA's AOC (Aircraft Operations Center) ground station infrastructure. The diagram specifies antenna installation, power distribution, control computer connectivity, and RF cabling for a 430 MHz monopole antenna system with optional band pass filtering for interference mitigation.

## Technical Approach
- **Base Station Configuration**: Ground-based control and data reception station
- **Antenna System**: AVMP430H SCall-Denke monopole antenna operating at 430 MHz
- **Power Distribution**: Dual 115V rack power inputs (primary and optional redundant)
- **Data/Control Links**: 
  - WiFi link from UAS to control computer
  - USB-to-Ethernet dongle for connectivity
  - GPS feed (iWRAP protocol) from antenna system
  - Optional external GPS input
- **RF Filtering**: ZVBP-435-S+ band pass filter (optional, added 06/10/2025 for AVAPS interference mitigation)
- **Location Standard**: Components mounted at Station 7 (STA 7)

## Products & Capabilities Described

### Black Swift UAS
- Integrates with NOAA AOC ground infrastructure
- Equipped with 430 MHz communication/data link capability
- Supports WiFi telemetry and control

### AVMP430H Monopole Antenna (430 MHz)
- SCall-Denke brand monopole antenna
- Operates at 430 MHz frequency band
- Requires installation per report RP-701139
- Provides GPS feed via iWRAP protocol

### Supporting Components
- **RG-400 RF Cable**: As needed quantity
- **Band Pass Filter (ZVBP-435-S+)**: Optional, used to address AVAPS interference issues
- **USB-to-Ethernet Dongle**: Connectivity interface
- **Control Computer**: User-supplied laptop/tablet powered by external supply

## Use Cases & Applications
NOAA aircraft operations monitoring and reconnaissance missions via UAS integration with AOC ground station infrastructure.
