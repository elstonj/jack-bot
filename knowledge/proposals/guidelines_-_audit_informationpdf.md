## Executive Summary
This is a procedural guidelines document for NASA SBIR applicants regarding disclosure and documentation of federal audits of accounting systems. It outlines what information must be provided and by whom when responding to audit-related questions on NASA SBIR Phase II proposals.

## Content Summary
This is not a technical proposal or report, but rather a template/guidelines document instructing firms on how to complete the "Audit Information" section of a NASA SBIR Phase II application form. It contains no substantive information about Black Swift Technologies' capabilities, products, or technical approaches.

## Key Administrative Requirements Listed
- Only the designated Firm Admin can update audit information
- Must disclose if federal agencies have audited accounting system
- If audited, must provide: auditing agency name, office location, POC details
- Must indicate if accounting system was approved by auditing agency
- Must disclose any rate agreements negotiated with auditing agency
- Must disclose if overhead/cost audits were performed with dates
- Must upload audit documentation (max 2 MB, PDF/DOC/DOCX format)
- Must explain if rates other than audit agreement rates are used in proposals
