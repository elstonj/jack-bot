## Executive Summary
Black Swift Technologies develops advanced avionics and autonomous flight control systems for small UAS, with a focus on data-centric operations and on-board artificial intelligence for fault detection. The company's core differentiators are the SwiftCore™ Flight Management System for autonomous payload-focused control and a machine learning-based system for predicting and mitigating aircraft failures before they occur.

## Technical Approach

**SwiftCore™ Flight Management System:**
- Payload-focused control system enabling real-time telemetry and control through autopilot data link
- Autonomously modifies flight paths based on sensor inputs to minimize operator workload
- End-to-end avionics solution enabling fully autonomous UAS operations
- Addresses key data capture dependencies: image overlap density/consistency, platform connectivity, flight planning/execution ease
- Implements machine learning to achieve fixed-wing aircraft longevity with multirotor simplicity

**On-Board Augmented Intelligence (AI Systems):**
- Hardware and firmware for individual subsystems with integrated sensor and machine learning algorithms
- Uses regression and classification techniques to identify in-flight failures and track maintenance requirements
- Performs automatic mitigation of failures and guides users through failures with "ECAM-like" system interface
- Enables autonomous response to propulsion failures, including remote/autonomous landings

## Products & Capabilities Described

**S2 UAS:**
- Features modular field-swappable payload system
- Payload positioned in nose cone in front of propeller for uncontaminated atmospheric measurements
- Rapid field payload changes using common power, data, and mechanical interface (no specialized tools required)
- Enables in-field payload calibration separate from aircraft

**SwiftCore™ Flight Management System:**
- Real-time telemetry and control via autopilot data link
- Sensor-based fully autonomous flight capability
- Advanced end-to-end avionics solution for control, communication, and commanding of UAS

**Modular Payload System:**
- Standardized interface for rapid integration of new sensing payloads (lab or field)
- Applications demonstrated include:
  - Soil Moisture Mapping (L-band radiometer)
  - Wildfire Monitoring (CO₂, CO, aerosol, RH, pressure, temperature, trace gas sensors, multispectral camera)
  - Multi-Angular Remote Sensing (531nm band monitor, Tetracam Mini-Multiple Camera Array, multispectral camera)
  - Volcano Monitoring (CO₂ trace gas sensor, nephelometer, temperature, pressure, humidity, wind sensors)
