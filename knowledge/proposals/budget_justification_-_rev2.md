# Budget Justification - Rev. 2

## Document Metadata
- Type: Budget justification for proposal revision
- Client/Agency: NASA
- Program/Solicitation: NASA Balloon Observations (implied from folder structure)
- Date: 2025-06-27 (document creation/modification date)

- Key Personnel: Jack Elston (Principal Investigator), Maciej Stachura (Principal Engineer/Aerospace Engineer)

## Executive Summary
This budget revision justifies increased costs (from ~$43K to ~$61K in direct labor) for a NASA balloon observation project, primarily driven by additional engineering support for platform integration with Urban Sky's balloon system, ground system modifications, and flight testing. The project includes procurement of a Black Swift S0 UAS and integration support with an external balloon platform subcontractor.

## Technical Approach
- **PI/Principal Engineer Focus**: ~350 hours for technical direction, interface design, integration oversight, and stakeholder coordination (Jack Elston/Maciej Stachura)
- **Aerospace/Avionics Engineer**: ~300 hours for electrical interface prototyping, launch command system integration, and balloon system simulation testing
- **Key Integration Work**: 
  - 80 hours for Urban Sky Risk Reduction Flight Test platform integration
  - 40 hours for ground system modifications
  - 80 hours for balloon system integration related to test flight
- **Timeline**: PI allocated ~2 months; additional engineer for ~1.75 months

## Products & Capabilities Described

### S0 Aircraft (UAS)
- **What it is**: Black Swift S0 unmanned aerial system with integrated sensor suite
- **Specifications**: 
  - Sensor suite includes: 3D winds measurement, pressure, temperature, humidity, surface temperature, terrain/wave height measurement
  - Integrated avionics for balloon-launch support
- **Proposed Use**: Full system validation, limited flight testing, demonstration
- **Budget Allocation**: $18,000 for one fully equipped unit

## Use Cases & Applications
- **Balloon observation missions**: Integration with Urban Sky's balloon platform for atmospheric data collection
- **Flight demonstration**: Regulatory preparation and flight demonstration activities
- **Risk reduction testing**: Urban Sky Risk Reduction Flight Test
