# A Machine Learning Software for Predicting and Improving UAS Maintenance Schedules

- Program/Solicitation: SBIR FX203-CSO1-1421
- Date: 2020 (Phase I submission; document modified May 28, 2021)

- Key Personnel: Dr. Jack S. Elston (Principal Investigator, CEO); Dr. Maciej Statura (CTO, Machine Learning Algorithm Design & Regulatory Expert)

## Executive Summary
Black Swift Technologies proposes developing a machine learning-based software system to analyze UAS flight logs and predict maintenance needs, preventing aircraft failures and reducing downtime. The system uses unsupervised learning for anomaly detection to build behavioral models of aircraft across varying mission conditions and alert operators when flights deviate from normal patterns. Phase I involves a feasibility study with the Air Force Academy's Center for Unmanned Aircraft Systems (CUAS) to adapt the software to USAF requirements.

## Technical Approach
- **Core Technology**: Unsupervised machine learning for anomaly detection applied to UAS telemetry data
- **Platform**: Web-based delivery system with red/yellow/green diagnostic dashboard designed for non-specialist technicians
- **Methodology**: Algorithms build predictive models of how aircraft should behave across diverse missions and flight conditions, then flag deviations indicating potential issues
- **Hardware Agnostic**: Designed to work across different UAS platforms
- **Monitoring Capabilities**:
  1. Wireless communication subsystem monitoring to prevent lost-link events
  2. Actuator and motor performance tracking for predictive maintenance recommendations
  3. Overall aircraft performance monitoring to detect environmental hazards (e.g., icing)
