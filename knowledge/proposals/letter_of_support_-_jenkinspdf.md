## Executive Summary
Dr. Nick Jenkins, a flight sciences engineer, expresses his support and commitment to participate as a subcontractor in Black Swift Technologies' NASA SBIR Phase II project to develop dynamic soaring capabilities for persistent Venus upper atmosphere observations. He will develop a six degree of freedom (6DOF) flight mechanics model validated through Earth-based flight testing before application to Venus operations.

## Technical Approach
Jenkins will develop a medium-to-high fidelity flight mechanics model with the following components:

**Aerodynamic Database Development:**
- Medium-fidelity aerodynamic database using Euler or panel methods
- Varied aircraft attitude, control surface deflections, and airspeeds
- Validation through Earth-based flight testing before Venus application

**6DOF Flight Mechanics Model Requirements:**
- Interface capability with existing servo-control models in Gazebo simulation environment
- Core modules for generating converged, validated temporal histories of flight scenarios and aircraft performance
- Pre- and post-processing modules for data visualization and analysis
- Utilization of flight test data (sounding, aircraft attitude, propeller pitch/speed, pilot control inputs, sensor data)
- Integration of total vehicle mass and inertial properties for both Earth test vehicle and Venus-destined vehicle

## Products & Capabilities Described
**BST UAV Platform:**
- A dynamic soaring-capable unmanned aerial system
- Designed for persistent observation operations in Venus upper atmosphere
- Will operate under Earth-based conditions first for validation before Venus deployment
- Requires integration with flight control systems and Gazebo simulation environment
