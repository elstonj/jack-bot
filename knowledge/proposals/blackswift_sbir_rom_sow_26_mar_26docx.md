## Executive Summary
This is a draft SOW for a Navy SBIR Phase II contract (N251-016, Ultra Maritime topic) in which Black Swift Technologies will develop and integrate a multi-sensor payload package onto its UAS platform. The work encompasses design, fabrication, testing, and analysis of acoustic and electric field sensing systems over a 30-month base period with anticipated funding of $300K, plus a 12-month option worth $120K.

## Technical Approach

### Base Effort (30 months, $300K, CPFF)
Black Swift will:
1. Provide design consultation to develop a payload sensing package based on concepts from a Phase I white paper
2. Develop physical architecture including:
   - Acoustic sensor design
   - Electric field sensors
   - Suspension system
   - Overall packaging and deployment scheme
   - Weight and power requirement estimates
3. Develop electronics and software designs:
   - Acoustic sensor and associated electronics
   - Electric field sensor and associated electronics
   - Data acquisition electronics
   - COTS computing/processing electronics
   - Data acquisition and ethernet link software
4. Fabricate two (2) sensor system units for UAS integration and testing
5. Provide engineering support for system-level testing at three locations: Ultra Maritime lab, Lapel testing site, and Black Swift site
6. Perform sensor package data analysis
7. Procure required materials
8. Provide design documentation and quarterly status reports

### Option Effort (12 months, $120K, CPFF)
- Support ongoing Black Swift test program with test support and data analysis
- Make minor design updates to sensor package
- Refurbish the two (2) sensor packages from base effort as required
- Procure material as needed
- Program review support and status reporting

## Products & Capabilities Described

**Black Swift UAS**
- The company's primary unmanned aerial system platform
- To be modified with integrated multi-sensor payload package
- Will host acoustic sensors, electric field sensors, data acquisition electronics, and processing systems

**Sensor Package Suite**
- **Acoustic Sensor**: To be designed with dedicated electronics
- **Electric Field Sensors**: Multiple units with dedicated electronics
- **Data Acquisition System**: Custom electronics to collect and process data from both sensor types
- **Computing/Processing**: COTS-based electronics for onboard processing
- **Software**: Data acquisition and ethernet link software for sensor integration
