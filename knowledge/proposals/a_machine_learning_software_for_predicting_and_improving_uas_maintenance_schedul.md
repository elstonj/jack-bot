# A Machine Learning Software for Predicting and Improving UAS Maintenance Schedules

## Document Metadata
- **Type:** SBIR Phase I Proposal
- **Client/Agency:** U.S. Air Force (Center for Unmanned Aircraft Systems, Air Force Academy)
- **Program/Solicitation:** SBIR FY2020 (Proposal #FX203-CSO1-1421)
- **Date:** October 28, 2020
- **BST Products/Systems Referenced:** S2 aircraft, web-based ML software platform
- **Key Personnel:** Dr. Jack S. Elston (Principal Investigator, CEO), Dr. Maciej Statura (Chief Technology Officer, Machine Learning Algorithm Design & Regulatory Expert)

## Executive Summary
Black Swift Technologies proposes developing unsupervised machine learning algorithms to analyze UAS flight logs and predict maintenance needs, reducing unplanned failures and increasing operational availability. The Phase I effort involves a feasibility study and cost analysis in collaboration with the Air Force Academy's Center for Unmanned Aircraft Systems to adapt BST's existing web-based ML platform for USAF assets, with potential Phase II prototype development and broader Air Force adoption.

## Technical Approach
- **Core Technology:** Unsupervised machine learning for anomaly detection applied to UAS flight telemetry data
- **Platform:** Web-based delivery system with red/yellow/green diagnostic dashboard designed for accessibility to non-specialist technicians
- **Algorithm Approach:** Build behavioral models of how aircraft should perform across varied missions and flight conditions; identify flights that deviate from expected performance patterns
- **Key Monitoring Functions:**
  1. Wireless communication subsystem monitoring to prevent lost-link events
  2. Automatic actuator and motor performance tracking to recommend preventive maintenance
  3. Overall aircraft performance monitoring to detect environmental hazards (e.g., icing conditions)
- **Planned Adaptations:** Plug-and-play compatibility with different drone platforms; integration with camera systems; consideration of cybersecurity requirements
