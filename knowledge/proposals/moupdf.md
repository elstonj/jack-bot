## Executive Summary
AFRL/RQQC commits to partnering with Black Swift Technologies on a Phase II SBIR effort to develop artificial intelligence algorithms for predictive maintenance, fault detection, and flight envelope monitoring of unmanned aerial systems (UAS). The project will adapt BST's AI tools (previously developed under NASA SBIR efforts) to the Cloud Cap Technology Piccolo autopilot and demonstrate capabilities on AFRL Group 1, 2, and 3 UAS platforms. Expected execution period: August 2021 to November 2022.

## Technical Approach

BST will develop a cloud-based software portal for automatic anomaly detection and predictive maintenance with the following components:

1. **Log File Processing**: Ingest flight logs, maintenance records, and failure data from Group 1 and Group 2 UAS into an automated framework; display analyzed data on cloud-based server solution with AFRL compliance for security and storage.

2. **Machine Learning Network Architecture**: Modify BST's current log server to incorporate a new machine learning network architecture (identified during Phase I) for improved fault detection performance. Create web-based portal demonstrating algorithm performance compared to previous iterations, with focus on minimizing false positives and false negatives.

3. **Performance Envelope Tracking**: Develop and document open APIs for both BST and AFRL to use for future algorithm development. Include real-time performance envelope evaluation and flight demonstration on the S2 UAS.

4. **S2 UAS Certification**: Provide necessary documentation and certification to enable S2 UAS approval for continued preventative maintenance and performance envelope analysis work.

## Products & Capabilities Described

### S2 UAS
- **What it is**: A Group 1-4 fixed-wing/hybrid VTOL unmanned aerial system operated by AFRL/RQQC
- **Proposed use**: Flight test platform for demonstrating preventative maintenance framework, performance envelope tracking, and real-time fault detection capabilities
- **Requirements**: Will require certification documentation, open API development, and flight demonstration of envelope monitoring

### S0 UAS
- **What it is**: Another AFRL-operated UAS platform (type/group not specified in document)
- **Proposed use**: Final integrated test platform for the system; at least one AF end-user will be trained to operate with the adapted solution

### Cloud-Based Server Solution
- **What it is**: BST's existing cloud platform modified to incorporate new ML architecture
- **Capabilities**: Ingest flight logs, display analyzed data, host web-based portal, provide real-time anomaly detection
- **Requirements**: Must comply with AFRL cloud security and storage standards

### Artificial Intelligence Algorithms
- **What it is**: Predictive maintenance and fault detection algorithms leveraging generative adversarial networks (GANs) and machine learning
- **Proposed use**: Real-time flight envelope monitoring, platform readiness assessment, anomaly detection across subsystems
- **Prior work**: Developed and matured under prior SBIR efforts with NASA

## Use Cases & Applications

1. **Attritable UAS Operations**: Real-time flight envelope performance information allows commanders to reason about tradeoffs between mission success and platform loss risk.

2. **Manned-Unmanned Teaming**: Envelope monitoring and estimation augments autonomous contingency planning, reducing pilot workload for manned teammates.

3. **Autonomous Platform Readiness**: Informs both commanders and autonomous decision-making algorithms of platform readiness, capability, and survivability in real-time.

4. **Preventative Maintenance Tracking**: Reduces manpower burden for operating and maintaining fielded systems; enables predictive rather than corrective maintenance.

5. **Air Launched Operations**: ALOBO (Air Launched Offboard Operations) program identified as particular transition target; BST already supporting air-launched UAS programs for NOAA.

6. **AI-Assisted Autonomy**: Skyborg Vanguard program would benefit from real-time fault detection, envelope estimation, and maintenance identification.

- AFRL/RQQC owns and operates multiple Group 1, 2, and 3 fixed-wing and hybrid VTOL UAS platforms that directly benefit from improved maintenance tracking
- Mission impact includes increased UAS squadron readiness, reduced manpower for maintenance and inspection, and increased mission effectiveness
