# Letter from State of Colorado - NASA SBIR Soil Moisture Project Support

## Document Metadata
- Type: Letter of commitment/support
- Client/Agency: State of Colorado
- Program/Solicitation: NASA SBIR Phase II-E, Contract NNX14CG09C
- Date: December 10, 2015
- BST Products/Systems Referenced: SuperSwift aircraft, Soil Moisture Mapping sUAS
- Key Personnel: Jack Elston (CEO, Black Swift Technologies)

## Executive Summary
The State of Colorado commits $125,000 in matching funding to support NASA SBIR Phase II-E contract NNX14CG09C for the Soil Moisture Mapping sUAS project. The state funding will support technical improvements to reduce electromagnetic interference (EMI) and commercialization efforts to enable broader adoption of the technology.

## Technical Approach
The Phase II-E proposal addresses electromagnetic interference (EMI) issues affecting the soil moisture sensor system through two parallel efforts:
1. **Aircraft modifications**: Improvements to the SuperSwift aircraft to reduce EMI generated by the electric propulsion system
2. **Electronics redesign**: Modifications to sensing electronics to reduce sensitivity to both environmental and on-board EMI sources

## Products & Capabilities Described
**Soil Moisture Mapping sUAS (small Unmanned Aircraft System)**
- NASA SBIR Phase II-E project under contract NNX14CG09C
- Equipped with soil moisture sensing capability
- Planned to be deployed on SuperSwift aircraft platform
- Key limitation identified: EMI interference affecting sensor performance

**SuperSwift Aircraft**
- Electric-powered sUAS platform
- EMI source from propulsion system requiring mitigation
- Subject to modifications to improve sensor compatibility

## Use Cases & Applications
- Soil moisture mapping for Colorado State priorities
- Broader utilization anticipated following EMI reduction and commercialization efforts
