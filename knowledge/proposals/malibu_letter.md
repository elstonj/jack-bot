## Executive Summary
NASA GSFC's MALIBU (Multi AngLe Imaging Bidirectional Reflectance Distribution Function) project commits $90,677 in direct funding to Black Swift Technologies to further develop the SuperSwift aircraft platform for integration with MALIBU's instrument package. The funding will support payload tray modifications, camera array development, and flight validation, with expected operations beginning summer 2017.

## Technical Approach
- Enhance SuperSwift sUAS platform for MALIBU instrument package integration
- Develop special features for swappable SuperSwift payload tray to enable quick field payload exchanges
- Build new camera array for MALIBU payload integration
- Conduct validation flight campaigns
- Leverage modular platform design to support multiple scientific missions simultaneously (soil moisture mapping + MALIBU reflectance measurements)

## Products & Capabilities Described

**SuperSwift Aircraft**
- Developed during Phase II of NASA SBIR contract NNX14CG09C
- Modular sUAS platform designed for soil moisture mission and broader scientific applications
- Key innovation: swappable payload tray enabling rapid field payload exchanges
- Expected to achieve TRL 9 (Technology Readiness Level) and MRL 7 (Manufacturing Readiness Level) by conclusion of CCRPP grant

## Use Cases & Applications
- **MALIBU Project**: Integration with Multi AngLe Imaging Bidirectional Reflectance Distribution Function instrument package for satellite calibration and validation
- **Soil Moisture Mapping**: Concurrent soil moisture measurement capability during MALIBU flights
- Broader scientific mission support via modular payload platform
