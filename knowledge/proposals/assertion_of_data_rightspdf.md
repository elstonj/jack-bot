# Assertion of Data Rights

## Document Metadata
- Type: Data Rights Assertion (Commercial Items - Technical Data and Computer Software)
- Client/Agency: U.S. Navy
- Program/Solicitation: Navy STTR Phase II - Hazardous Weather
- Date: 2026-02-25
- BST Products/Systems Referenced: SwiftPilot, SwiftStation, SwiftTab, SwiftFlow probe
- Key Personnel: Dr. Jack Elston (CEO, Black Swift Technologies, LLC)

## Executive Summary
This document asserts Black Swift Technologies' proprietary data rights over five core software and hardware systems developed for the Navy STTR Hazardous Weather program. All systems are claimed as commercial items developed prior to this project with partial private expense funding, giving BST restricted data rights under SBIR regulations.

## Products & Capabilities Described

**SwiftPilot** - Autopilot system
- Integrates 3D wind sensing and atmospheric sensing capabilities
- Status: Developed prior to this project; developed partially at private expense

**SwiftStation** - Control Station
- Ground control software for SwiftPilot autopilot operation
- Status: Developed prior to this project; developed partially at private expense

**SwiftTab** - User Interface
- Tablet/device-based UI for controlling UAS equipped with SwiftPilot
- Status: Developed prior to this project; developed partially at private expense

**SwiftFlow Probe** - 5-hole atmospheric probe
- Measures 3D wind data
- Specifically designed to function in precipitation conditions
- Status: Developed prior to this project; developed partially at private expense

**Web-based Communications, Terrain and Wind-Aware Planning**
- Mission planning software with environmental awareness
- Status: Developed prior to this project; developed partially at private expense

## Notable Details
- All five systems are asserted as commercial items with restricted data rights
- All were developed prior to the Navy STTR Phase II project, establishing pre-existing IP
- All were partially funded through private BST investment, strengthening IP position
- Dr. Jack Elston asserts rights on behalf of BST for all listed items
- Rights category cited: "Rights in SBIR data"
