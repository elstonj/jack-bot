## Executive Summary
This is a Navy STTR Phase II cost proposal for a collaborative project between Black Swift Technologies and Embry-Riddle Aeronautical University to develop expandable air-sea profiling capabilities using small uncrewed aircraft systems for hazardous weather observations. The proposal outlines 24 months of base work plus a 24-month option period, totaling $333,503 in funding.

## Technical Approach
- Build on Phase I results, including implementation of a turbulence algorithm developed in Phase 1
- Integration of S0 UAS data into meteorological/oceanographic models
- Analysis of field data and algorithm performance
- Development of wave height algorithms
- Testing upgrades to the S0 system resulting from Phase II development
- Optional phase includes analysis of extreme cold environment performance and over-water validation testing
- Final deliverables include concepts of operation for S0 usage in hazardous weather

## Products & Capabilities Described

### S0 UAS (Small Uncrewed Aircraft System)
- Expandable platform for air-sea profiling observations
- Intended for operations in hazardous weather conditions
- Being upgraded/modified through this Phase II project
- Expected to operate in extreme cold environments (option phase focus)
- Provides data suitable for integration into atmospheric/oceanographic models

## Use Cases & Applications
- Air-sea profiling in hazardous weather conditions
- Extreme cold environment operations
- Over-water measurements and validation
- Meteorological and oceanographic data collection
- Weather monitoring missions
