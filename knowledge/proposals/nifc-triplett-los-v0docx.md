# NASA Phase II SBIR: Improved Wildfire Forecast Models - Letter of Support

- Client/Agency: NASA (Phase II SBIR)
- Program/Solicitation: NASA Phase II SBIR - Wildfire Forecast Models
- Date: November 21, 2023
- BST Products/Systems Referenced: Uncrewed Aircraft System (UAS), meteorological sensor payload
- Key Personnel: Dr. Jack Elston (CEO, BST); Sean Triplett (NIFC, supporting letter writer)

## Executive Summary
This is a letter of support from Sean Triplett (Branch Leader for Tools and Technology at the National Interagency Fire Center) confirming participation in BST's NASA Phase II SBIR project. The effort proposes using a UAS-mounted meteorological sensor payload to collect data that will refine weather models for wildfire forecasting, with potential real-time distribution to firefighters via systems like ATAK.

## Technical Approach
- Deploy a meteorological sensor payload on a BST-developed UAS to collect atmospheric data
- Deliver collected data to the National Center for Atmospheric Research (NCAR) to reduce uncertainty in fire weather models used for operational forecasting
- Potentially integrate real-time data into situational awareness systems (e.g., ATAK) for deployed firefighters
- Research supports both NASA initiatives (FireSense Project, ACERO) and federal/state fire response agencies (USFS, DOI, state/local teams)

## Products & Capabilities Described
**BST UAS with Meteorological Sensor Payload**
- Purpose: Collect atmospheric/meteorological data over wildfire zones
- Application: Improve fire weather forecasting accuracy
- Integration potential: Real-time data feed to firefighter command systems

## Use Cases & Applications
- Wildland fire suppression planning and response
- Fire weather forecasting model refinement
- Real-time situational awareness for deployed firefighting personnel
- Support for emergency response operations
