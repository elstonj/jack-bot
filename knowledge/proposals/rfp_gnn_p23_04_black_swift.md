## Executive Summary
This RFP solicits Black Swift Technologies as a subcontractor to Barron Associates on a Phase II SBIR contract for developing Graph Neural Networks (GNN) for collaborative autonomous agent control in unmanned systems. BST's role focuses on technology commercialization, UAS platform integration, flight testing, and transition path development over 18 months with an $80,000 budget and estimated start date of June 1, 2023.

## Technical Approach
The overarching program aims to develop an AI GNN framework enabling collaboration between swarming unmanned aerial and ground systems (UxS). Key technical challenges being addressed:
- **Dynamic graph topology:** Current GNN approaches assume fixed graph structures; the solution must handle agents being added or removed from the swarm without retraining
- **Multi-vehicle heterogeneity:** Framework must work across different vehicle types (air and ground)
- **Robustness:** Must handle communication delays and data packet loss

**BST's specific technical role:**
- Lead integration of developed GNN software with a flight-tested UAS platform
- Support FAA Part 107 certification efforts for the test platform
- Provide technical feedback on integration characteristics that influence design (flight software, mission software, platform architecture)

## Products & Capabilities Described
- **UAS Platform:** BST to provide a selected flight vehicle for demonstrations and field testing; capable of supporting multi-agent swarm operations
- **Flight/Mission Software:** BST systems identified as potential integration platforms for the GNN framework
- **FAA Part 107 Pilot Support:** BST to provide certified pilot(s) for flight test operations

## Use Cases & Applications
- **Collaborative Unmanned Swarm Operations:** Multiple aerial and/or ground vehicles operating as coordinated agents
- **Field Demonstrations:** Two 5-day test periods at Virginia test facilities with final demonstration location TBD
- **Military/Defense Application:** US Army focus on autonomous agent collaboration for combat capabilities
