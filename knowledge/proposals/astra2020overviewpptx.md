## Executive Summary
ASTRA is Black Swift Technologies' space weather and ionospheric monitoring program built on core competencies in space systems, instrument development, data assimilation, and analysis. The program integrates space-based and ground-based data sources with multiple instruments and modeling tools to support space weather monitoring and Space Situational Awareness (SSA).

## Technical Approach
ASTRA integrates three domains:
- **Space Weather Modeling**: Uses TIMEGCM (Time-dependent Ionospheric Model), IDA4D (data assimilation), and AMIE (high-latitude electrodynamics models)
- **Instrument Development**: Scanning photometer systems, Fabry-Perot interferometer, temperature sensors, balloon dosimeters, space-based GPS receivers
- **Data Assimilation & Analysis**: Combines space-based GPS data with ground-based observations for comprehensive ionospheric characterization

## Products & Capabilities Described

### CASES (Connected Autonomous Space Environment Sensor)
- **What it is**: Dual-frequency GPS software receiver designed for robust space weather monitoring in scintillating environments
- **Capabilities**: 
  - Navigation solution PLUS space weather monitoring
  - Measures TEC (Total Electron Content) and scintillation parameters
  - Designed for operation in challenging ionospheric conditions
- **Status**: Over 20 systems deployed; reliable operation confirmed; ready for sale
- **Form Factors**: Two configurations offered—standard version and Antarctic/CubeSat version
- **Connectivity Options**:
  - Local storage: SDHC, USB, RS232
  - Local network: WIFI (standard config only), Ethernet
  - Remote network: Internet connectivity with WAN capability
  - Real-time internet operation supported
