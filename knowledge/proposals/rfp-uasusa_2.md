# RFP-UASUSA 2: Virtual Redundancy for Safety Assurance in the Presence of Sensor Failures

- Type: Request for Proposal (RFP)
- Client/Agency: UASUSA (Unmanned Aircraft Systems USA), Longmont, CO
- Program/Solicitation: RFP #P14-53-02; Army Phase I subcontract (Barron Associates prime)
- Date: December 12, 2014 (submission deadline: December 16, 2014)
- BST Products/Systems Referenced: Tempest flight control system
- Key Personnel: Jack Elston (last editor); Technical contact: Alec Bateman (Barron Associates); Administrative contacts: Bruce Wilson, Connie Hoover

## Executive Summary
This RFP solicits experimental and flight test work to develop sensor failure data and test plans for the Tempest flight control system, with focus on virtual redundancy approaches to maintain safety assurance when sensors fail. The 24-month Phase II effort is budgeted at $15,000 with 90% of work concentrated in Year 2.

## Technical Approach
The work is divided into three main tasks:

1. **Experimental Sensor Failure Data Collection**
   - Identify subset of Tempest flight control system sensors suitable for induced failure testing in laboratory
   - Develop experimental system to replicate relevant flight environment conditions for sensor failure induction
   - Conduct controlled sensor failure experiments and deliver resulting data to Barron Associates

2. **Sensor Failure Flight Test Planning**
   - Identify Tempest sensors capable of realistic in-flight failure induction
   - Design experimental system for inducing these failures in actual flight environment
   - Develop comprehensive flight test plan for sensor failure validation

3. **Program Management**
   - Document technical work and support interim/final NASA reporting by Barron Associates
