# Black Swift: A Machine Learning Software for Predicting and Improving UAS Maintenance Schedules

## Document Metadata
- **Type:** SBIR Phase I Proposal Summary
- **Client/Agency:** U.S. Air Force (USAF)
- **Program/Solicitation:** DoD SBIR X20.3 Solicitation
- **Proposal Number:** 20-1-FX203-CSO1-1421
- **Date:** October 28, 2020
- **BST Products/Systems Referenced:** Avionics monitoring system, proprietary avionics with monitoring nodes
- **Key Personnel:** Not specified in summary

## Executive Summary
Black Swift Technologies proposed a 3-month Phase I feasibility study to develop a machine learning-based predictive maintenance solution for small UAS platforms operating within the USAF. The approach would leverage unsupervised ML algorithms on existing USAF avionics data and BST's proprietary monitoring nodes to provide early warning diagnostics of critical system failures, improving maintenance scheduling and reducing downtime.

## Technical Approach
- **Baseline:** BST's current avionics monitoring system
- **Core Technology:** Unsupervised machine learning algorithms for early warning and diagnostics of potential critical system failures on small UAS
- **Data Collection:** 
  - Primary source: Avionics data already collected by USAF
  - Backup capability: BST-developed monitoring nodes that can be installed aboard candidate platforms to supplement datasets if USAF data proves insufficient
- **Implementation:** Real-time ML algorithm analysis and feedback capability
- **Processing Location:** Algorithm work would be performed on BST systems while obtaining data from USAF sources
- **Optionality:** Stakeholders could request real-time data gathering from their UAS vehicles using BST's monitoring capabilities

## Products & Capabilities Described

### Avionics Monitoring System
- BST's baseline technology for gathering flight data

### Proprietary Monitoring Nodes
- BST-developed monitoring hardware

### ML-Based Preventative Maintenance Solution
- Uses unsupervised learning algorithms
- Provides early warning of critical system failures
- Generates diagnostic information
- Delivers real-time analysis and feedback
