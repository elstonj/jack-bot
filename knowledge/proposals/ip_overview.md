## Executive Summary
This document outlines BST's intellectual property and technological advantages for methane detection using fixed-wing UAS. BST has partnered with Quanta3 to provide the world's lightest methane sensor with sub-2 ppb detection capability, combined with proprietary real-time flight plan modification and on-board wind speed measurement capabilities unique to their platform.

## Technical Approach
- Partnership with Quanta3 (leading US methane detection technology provider) for joint development and exclusive provider arrangement
- Integration of ultra-lightweight methane sensor onto fixed-wing UAS platform
- Real-time, on-board wind speed measurement system integrated into UAS
- Real-time flight plan modification capability triggered by methane detection
- Leverages BST's proven BVLOS (beyond visual line of sight) operational expertise

## Products & Capabilities Described

### Methane Detection Sensor
- **What it is:** World's lightest methane sensor with <2 ppb (parts per billion) detection level
- **Specifications:** Detection threshold below 2 ppb; minimal weight for UAS integration
- **Development:** Joint development with Quanta3; exclusive provider arrangement
- **Use:** Fixed-wing UAS methane detection missions

### "Follow the Scent" Sensor-Based Navigation
- **What it is:** Proprietary flight planning system that modifies flight paths in real-time based on methane detection
- **Capability:** Only UAS capable of this real-time flight plan modification based on sensor input
- **Function:** Allows autonomous adaptation to plume presence for improved surveying efficiency

### On-Board Wind Speed Measurement
- **What it is:** Real-time, on-board wind speed measurement system
- **Capability:** Accurately measures absolute wind speed at the UAS platform
- **Critical function:** Essential for precise leak rate characterization in methane detection missions
- **Uniqueness:** Only UAS platform with this integrated capability

### BVLOS Operations
- **What it is:** Beyond visual line of sight operational capability
- **Status:** Proven, unmatched expertise
- **Significance:** Enables extended-range methane detection surveys

## Use Cases & Applications
- Methane leak detection and localization
- Leak rate characterization (enabled by wind speed measurement)
- Extended-range surveying beyond visual line of sight
- Real-time adaptive surveying (following methane plumes)
