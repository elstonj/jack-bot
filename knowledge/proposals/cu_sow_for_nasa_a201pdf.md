# CU SoW for NASA A2.01 (GPS-Denied Navigation)

## Document Metadata
- Type: Statement of Work (Subcontract)
- Client/Agency: NASA
- Program/Solicitation: Small Business Innovation Research (SBIR) Phase 1
- Date: 03/03/2025
- BST Products/Systems Referenced: Uncrewed aircraft system (unspecified model)
- Key Personnel: Dr. Christoffer Heckman (University of Colorado Boulder, Associate Professor, Department of Computer Science); Beck Cotter (BST, last editor)

## Executive Summary
Black Swift Technologies partnered with Dr. Christoffer Heckman at the University of Colorado Boulder to develop terrain-aware navigation and GPS-denied navigation capabilities for a NASA SBIR Phase 1 project. Dr. Heckman will provide algorithm expertise and technical guidance while assisting BST in transitioning the work to a Phase 2 proposal.

## Technical Approach
- Development of technical approach and algorithms for terrain-aware navigation and GPS-denied capability using the same sensor suite and hardware
- Focus on sensor selection, algorithm selection and customization, and integration with an existing uncrewed aircraft system
- Dr. Heckman to provide expertise in algorithm development and technical direction based on his specialization in this area

## Products & Capabilities Described

**GPS-Denied Visual Navigation**
- Capability for aircraft navigation without GPS signal
- Implementation using existing technologies and sensor integration

**Terrain Relative Flight and Obstacle Avoidance**
- Navigation relative to terrain features
- Real-time obstacle detection and avoidance capability
- Integration with uncrewed aircraft system

## Use Cases & Applications
- GPS-denied operations (applicable to GPS-denied environments, potential military/contested environments)
- Terrain-relative navigation missions
- Obstacle avoidance in autonomous flight operations

## Key Tasks
1. **Technical Approach Development**: Algorithm selection, sensor selection, hardware integration planning, and implementation roadmap
2. **Phase 2 Proposal Development**: Contributing qualifications, expertise documentation, laboratory resources description, technical sections, and plan of work
