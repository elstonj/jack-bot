## Executive Summary
This document outlines BST's work tasks as a subcontractor to Barron Associates on an Army Phase I project. The work focuses on generating experimental sensor failure data and developing flight test plans for the Tempest flight control system, with a budget target of $15,000.

## Technical Approach
BST's approach consists of three main task areas:

1. **Experimental Sensor Failure Data**: Identify a subset of Tempest flight control system sensors suitable for failure testing, develop an experimental system to induce realistic failures in a controlled environment that replicates flight conditions, and conduct the failure experiments to deliver data to Barron Associates.

2. **Sensor Failure Flight Test Plan**: Identify sensors suitable for in-flight failure testing, design a system for inducing those failures in an actual flight environment, and develop a comprehensive flight test plan.

3. **Program Management**: Provide technical documentation to support Barron Associates' interim and final reports to NASA.

## Products & Capabilities Described
- **Tempest Flight Control System**: UAS flight control system with multiple sensors; BST was tasked with stress-testing sensor redundancy and failure modes through both ground-based and flight testing.

## Use Cases & Applications
- Autonomous system reliability and robustness validation
- Sensor fault tolerance evaluation for unmanned aircraft systems

## Notable Details
- Work structured as subcontract to Barron Associates (prime contractor)
- Coordination requirement with Barron Associates on sensor/failure selection to align with project goals
- Deliverables include raw experimental data and a flight test plan document
- Modest budget suggests focused, targeted sensor evaluation work rather than comprehensive system redesign
