# Budget - TX2_Carrier_and_Cabling

## Document Metadata
- Type: Purchase Order/Invoice
- Client/Agency: Black Swift Technologies (BST)
- Program/Solicitation: NASA 2018 SBIR - Terrain Following (Phase II)
- Date: December 18, 2018
- Vendor: WDL Systems LLC
- Key Personnel: Jack Elston, Austin Anderson (BST)

## Executive Summary
This is a purchase order for embedded computing carrier hardware and cabling components from WDL Systems (CTI Spacely product line) to support Black Swift Technologies' NASA SBIR Phase II project on terrain following. The order totals $955.00 and includes a TX2 carrier board, power supply, and specialized cabling for integration with autopilot systems.

## Products Ordered

### 1. CTI Spacely 60W Multi-Region AC/DC Power Supply (MSG080)
- **Cost:** $85.00
- **Specifications:** Desktop style, 12V 60W output, RoHS compliant
- **Purpose:** Power supply for the TX2 carrier system

### 2. CTI Spacely Cable Kit (CKG045)
- **Cost:** $97.00
- **Components:**
  - 2x CBG116 System Cable (unterminated wires to 20-pin MiniTek w/Latch)
  - 2x CBG117 RJ-45 to Minitek Cable
  - 1x CBG104 Dual USB 2.0 panel mount to 8-pin MiniTek w/Latch
  - 1x CBG191 DB9 to 10-pin Header 2mm
- **Purpose:** Interconnection cabling for system integration

### 3. CTI Spacely Carrier for TX2/TX1 (ASG006)
- **Cost:** $758.00 (primary component)
- **Key Specifications:**
  - Support for up to 6 MIPI CSI-2 interfaces
  - Pixhawk Autopilot-compatible IO
  - 2x Gigabit Ethernet
  - 1x microHDMI
  - 2x USB 3.0, 2x USB 2.0, 1x USB OTG
  - 1x miniPCIe slot
  - mSATA storage slot
  - CAN 2.0b port
  - 2x UART, I2C, SPI
  - 14 GPIO (all at +3.3V IO)
  - Optional GPS/GNSS module support
  - Operating temperature: -40°C to +85°C
  - RoHS compliant
- **Purpose:** Main computing carrier board for NVIDIA TX2 or TX1 processor with specialized I/O for autonomous systems integration

## Notable Details
- All components are RoHS compliant and rated for wide temperature ranges (-40°C to +85°C), suitable for airborne operations
- The carrier board's Pixhawk autopilot integration indicates this system was designed for autonomous aerial vehicle applications
- Multiple camera interfaces (MIPI CSI-2) support multi-sensor configurations typical of terrain-mapping systems
- Redundant networking (2x GbE) and extensive I/O suggest a system designed for complex sensor integration and real-time data processing
- This appears to be component procurement for building a terrain-following capable autonomous system for the NASA SBIR project
