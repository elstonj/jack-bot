## Executive Summary
This document provides step-by-step internal procedures for Black Swift Technologies to submit deliverables and invoices for a Navy SBIR Phase I magnetometer contract (N68335-25-C-0492). It covers deliverable submission to the Navy's website, invoice creation in QuickBooks, and invoice submission through the WAWF/PIEE system.

## Process Overview

### Part 1: Navy Deliverable Submission

**Preparation (Step 1):**
- Check deliverable requirements 1-2 weeks before due date in Asana project brief or contract
- Apply CUI (Controlled Unclassified Information) markings: minimum CUI/Dist B for SBIR to protect vendor confidentiality
- Follow distribution statements per CDRL requirements
- File specifications:
  - Maximum 25MB file size
  - PDF format only
  - Naming convention: Full contract number + CLIN/Item Number (e.g., N0001412C9999A001ProgressReport1.pdf)

**Access & Upload Code (Step 2):**
- Upload Code for this contract: LXONII5161 (shared with project 550-2, N68335-25-C-0270 STTR Phase 1)
- Website: https://www.navysbirprogram.com/navydeliverables/default.aspx
- Login process: Firm Name (Black Swift Technologies) → Forgot Upload Code → Use Point of Contact email (Jack Elston: elstonj@blackswifttech.com)

**Submission Process (Step 3):**
- Log in to Navy deliverable website
- Click "Contract Deliverables" → "Add a new contract deliverable"
- Select correct contract number: N68335-25-C-0492
- Confirm CUI marking requirements (required for this contract)
- Select correct deliverable type matching CLIN number in contract
- Upload PDF file (one deliverable at a time; multiple deliverables submitted separately)
- Confirm file correctness before finalizing
- Repeat for additional deliverables as needed

### Part 2: Invoice Submission via WAWF/PIEE

**Preparation:**
- Create invoice in QuickBooks in advance for the specific CLIN being submitted (or download pre-created invoice)
- Access: https://piee.eb.mil/
- Two-factor authentication: Verification code sent to operations@blackswifttech.com (expires in 15 minutes)

**WAWF Document Creation:**
1. Select "WAWF" button under Payment
2. Click "Create Document" under Vendor section
3. Verify CAGE Code/DUNS is correct
4. Set "Contractual?" to "Y"
5. Enter Contract Number: N68335-25-C-0492
6. Select "From Template?" = "N"
7. Select appropriate CLIN number from contract
8. Select document type: "Invoice and Receiving Report (Combo)" (Note: Some reviewers may require "Invoice" instead; no consistent guidance provided)
9. Set Inspection: Destination, Acceptance: Destination

**Data Entry - Header Tab:**
- Invoice Number: Use QuickBooks-generated invoice number (for tracking only)
- Invoice Date: Use CLIN due date
- Final Invoice: "N" (unless absolute final invoice of contract)
- Shipment Number: Create dummy number
- Shipment Date: Date after CLIN due date (e.g., if CLIN due Dec 11, use Dec 12)
- Final Shipment: "N" (unless absolute final)
- Click "Save Draft Document"

**Data Entry - Line Item Tab:**
- Click "Edit" on the listed CLIN
- Set "Qty Shipped" to "1"
- Click "Save CLIN"
- Return to Header tab to verify no error messages

**Data Entry - Attachments Tab:**
- Recommended: Attach actual invoice PDF (optional but preferred by some reviewers)
- File name must have no spaces
- Select Attachment Type: "Invoices"
- Click "Save Draft Document"

**Submission & Notifications:**
- Return to Header tab and click "Submit"
- If error occurs about "Final Invoice already exists," go to Comments tab and note:
  - State that this is not a final invoice
  - Specify which CLIN is being submitted
- Upon successful submission, click "Send Additional Email Notifications"
- Add following recipients:
  - Lisa Moua (Contract Administrator): lisa.d.moua.civ@mail.mil
  - Sean Seremet (Navy Operations Analyst): sean.e.seremet.civ@us.navy.mil
  - Note: Sean Seremet reviews deliverables first before invoice approval; manual notification ensures awareness
- Click final Submit to send emails

## QuickBooks Invoice Creation

**Process:**
1. Navigate to Projects section
2. Select correct project (example given: [400-5] SBIR - SMM 22.4D Phase I)
3. Click "Add to Project" button → Select "Invoice"
4. Complete invoice form:
   - Product/Service: "Service: Custom Aerospace Engineering" (or per Jack Elston's guidance)
   - Description: Copy from contract CLIN description
   - Qty: 1
   - Rate: Dollar amount from contract for applicable CLIN
   - Tax: Ensure unchecked (required for government invoices)
   - Class: "Government"
5. Click "Save" (NOT "Save and Send")
6. Use generated invoice number in PIEE/WAWF submission
7. Invoice is for internal tracking only; never send from QuickBooks
