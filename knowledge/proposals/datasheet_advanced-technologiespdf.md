## Executive Summary
This datasheet describes Black Swift Technologies' advanced avionics and autonomy capabilities, focusing on the SwiftCore™ Flight Management System for data-centric autonomous flight control and on-board augmented intelligence using machine learning for fault detection and mitigation. The document emphasizes how these technologies enable reliable, autonomous UAS operations for scientific missions and support regulatory approval for BVLOS and fully autonomous operations.

## Technical Approach

**SwiftCore™ Flight Management System:**
- Payload-focused control system with real-time telemetry and control via autopilot data link
- Autonomously modifies flight paths based on sensor inputs to minimize operator workload
- End-to-end avionics solution enabling fully autonomous flight
- Data-centric design optimizing image overlap density, consistency, platform connectivity, and flight planning/execution

**On-Board Augmented Intelligence:**
- Machine learning algorithms (regression and classification techniques) for fault detection and mitigation
- Modular networked system for automated fault detection on critical aircraft subsystems
- Predicts system failures before occurrence
- Provides ECAM-like (Electronic Centralized Aircraft Monitoring) guidance through failures
- Automatic mitigation capability for propulsion failures requiring autonomous/remote landings
- Tracks maintenance requirements and in-flight failures

## Products & Capabilities Described

**SwiftCore™ Flight Management System:**
- Advanced avionics subsystem for small UAS
- Enables autonomous flight with sensor-based path modification
- Provides real-time pilot communication and command capability
- Minimizes operator workload while enhancing data quality

**S2 UAS:**
- Features highly autonomous flight capability (quoted as "absolutely brilliant" and "idiot-proof")
- Equipped with modular field-swappable payload system
- Sensor suite positioned in nose cone in front of propeller for clean, uncontaminated atmospheric measurements
- Payload swapping requires no specialized tools and uses common power, data, and mechanical interface
- Enables in-field payload calibration separate from aircraft

**AI Systems/On-Board Augmented Intelligence:**
- Hardware and firmware for individual subsystems with embedded sensor and ML algorithms
- Regression and classification techniques for subsystem tracking
- Automatic failure mitigation and ECAM-like user guidance
- Rapid response capability to propulsion failures

## Use Cases & Applications

The S2 with modular payloads has been configured for:
- **Soil Moisture Mapping**: L-band radiometer
- **Wildfire Monitoring**: CO₂, CO, aerosol sensors; RH, pressure, temperature sensors; trace gas sensors; multispectral camera
- **Multi-Angular Remote Sensing**: 531nm band monitor; Tetracam Mini-Multiple Camera Array Imaging System; multispectral camera
- **Volcano Monitoring**: CO₂ trace gas sensor; nephelometer; temperature, pressure, humidity, and wind sensors
- **Arctic/Alpine Research**: Referenced as operational environment
