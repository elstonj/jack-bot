## Document Purpose
This document provides standardized federal instructions for completing the "Research & Related Personal Data" form required for grants.gov submissions. It served as a reference document for BST's NOAA SBIR Phase II proposal.

## Content Summary
This is not a BST technical document but rather a government form instruction guide. It contains field-by-field instructions for the OMB Form 4040-0001, which collects biographical and demographic information about Project Directors/Principal Investigators (PD/PI) in federal grant applications.

## Form Fields Documented
The instructions detail completion requirements for:
- Name fields (prefix, first, middle, last, suffix)
- Gender (optional)
- Race categories (American Indian/Alaska Native, Asian, Black/African American, Native Hawaiian/Pacific Islander, White)
- Ethnicity (Hispanic/Latino)
- Disability status (Hearing, Visual, Mobility/Orthopedic, Other)
- Citizenship
- Multiple person entry capability

## Notable Details
- All demographic fields are marked as optional
- Form includes standard federal racial and ethnic categories
- Includes "Do Not Wish to Provide" options for sensitive demographic data
- This is a reference/instructional document, not a substantive proposal or technical document
