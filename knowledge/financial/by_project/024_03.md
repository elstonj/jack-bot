# Project 024-03 — Financial Summary

## Project Overview
- **Full Project Name**: "On-demand wildfire susceptibility from fuel moisture and vegetative index maps"
- **Client/Agency**: NASA (ROSES 2024, A.59 funding opportunity)
- **Contract Type**: NASA ROSES research grant (Step 2 submission)
- **Period of Performance**: April 1, 2025 – April 1, 2028 (3 years)
- **Brief Description of Scope**: Multi-institutional research project integrating small UAS, NISAR satellite, and UAVSAR data to generate high-resolution soil moisture and live fuel moisture maps for wildfire susceptibility assessment.
