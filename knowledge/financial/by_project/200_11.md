# Project 200-11 — Financial Summary

## Project Overview
- **Full Project Name:** Persistence Demo (NASA) — Demonstrating persistent IR measurements over wildfires using small, fixed-wing uncrewed aircraft
- **Client/Agency:** NASA Ames Research Center
- **Contract Type:** SBIR Phase III (Fixed Price)
- **Contract Number:** 80NSSC25CA043
- **Period of Performance:** January 2025 – September 2025 (9 months)
- **Project Status:** ARCHIVED / COMPLETED (11/24/2025)
- **Brief Description:** Demonstrate 24-hour persistent infrared (IR) measurements over wildfires using the S2 platform (small, fixed-wing uncrewed aircraft). The project included integration of Remote ID detection systems, NightFOX IR payload development, and a live 24-hour persistence demonstration at Pawnee National Grassland (August 27–28, 2025) with attendees from USFS, NASA, and USDA.
