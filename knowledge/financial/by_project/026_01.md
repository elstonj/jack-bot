## Key Deliverables & Milestones

**From Proposals & Reports:**

### Task B.2: S0 Measurement Evaluation
- **Scope:** Data validation, error analysis (wind, turbulence, height calculations), frequency response testing (100 Hz, 50 Hz, 25 Hz, 10 Hz)
- **Key Milestone:** De-icing strategy development
  - Heat Multi-Hole Pitot Probe (MHP) using PTC heating with thermostat control
  - Evaluate Ice-X de-icing compounds on wings/propellers
  - Propeller de-icing flagged as critical ("show stopper") issue
  - Potential NASA wind tunnel testing planned
- **Status:** Document last modified 2026-01-02; active development
- **Financial Implication:** Testing and NASA collaboration likely represent material cost; de-icing solution viability critical to Phase II progression

### Task B.5: S0 Weather Collection Concept of Operations (CONOPS)
- **Scope:** Define operational procedures for S0 Air-Deployed (S0-AD) deployment from U.S. Navy platforms (P-8A Poseidon, C-130, MH-60)
- **Key Components:**
  - Deployment sleeve with drag parachute
  - SwiftTab control tablet (real-time mission adjustment)
  - SwiftCore mission planning/data download system
  - Datalink integration (VHF-based)
- **Status:** Document created 2025-12-16, modified 2026-01-05; Phase I deliverable nearing completion
- **Financial Implication:** SwiftCore software development and platform integration likely underway; Phase II prep (DCAA survey completion noted in documents) indicates Phase II proposal anticipated
