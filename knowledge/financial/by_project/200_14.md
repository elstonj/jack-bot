# Project 200-14 — Financial Summary

## Project Overview
- **Project Name:** [200-14] Autonomy
- **Full Title:** Adaptive and Secure Autonomy for UAS: A Modular Approach to Flight Control, Machine Learning Integration, and Multi-Vehicle Coordination
- **Client/Agency:** NASA / Shared Services Center (Stennis Space Center, Mississippi)
- **Contract Type:** SBIR Phase I
- **Contract Number:** 80NSSC25C0155
- **Principal Investigator:** Dr. Jack Elston
- **Period of Performance:** September 29, 2025 – March 27, 2026 (6 months)
- **Project Status:** **ARCHIVED** — All work completed; final administrative tasks and payments received as of April 16, 2026

**Brief Description of Scope:**
Development of a modular, layered architecture for the SwiftCore avionics system enabling aircraft autonomy with multi-vehicle coordination, machine learning-based sensing and control, safe sandbox frameworks for experimental modules, and supervisory control with safety thresholds. Deliverables included 5-layer functional architecture, ML terrain-following module, plugin architecture integration, and comprehensive flight/bench testing.
