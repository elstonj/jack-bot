# Project 200-14 — Financial Summary

## Project Overview
- **Project Name:** [200-14] Autonomy
- **Full Title:** Adaptive and Secure Autonomy for UAS: A Modular Approach to Flight Control, Machine Learning Integration, and Multi-Vehicle Coordination
- **Client/Agency:** NASA (Shared Services Center, Stennis Space Center, MS)
- **Contract Type:** SBIR Phase I
- **Contract Number:** 80NSSC25C0155
- **Principal Investigator:** Dr. Jack Elston, Black Swift Technologies LLC
- **Period of Performance:** September 29, 2025 – March 27, 2026 (6 months)
- **Project Status:** **ARCHIVED** — All work completed; final administrative tasks and payment receipt closed April 16, 2026
- **Brief Description:** Development of a modular, layered architecture for the SwiftCore avionics system enabling aircraft autonomy with multi-vehicle coordination, machine learning-based sensing and control, safe sandbox framework for experimental modules, and supervisory controller with safety thresholds. Technical achievements include 5-layer functional architecture, ML terrain-following module with automated PID tuning, plugin architecture, and successful flight/bench testing.
