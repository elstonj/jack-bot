# Project 200-14 — Financial Summary

## Project Overview
- **Full Project Name:** Adaptive and Secure Autonomy for UAS: A Modular Approach to Flight Control, Machine Learning Integration, and Multi-Vehicle Coordination
- **Client/Agency:** NASA (Stennis Space Center, Mississippi)
- **Contract Type:** SBIR Phase I
- **Contract Number:** 80NSSC25C0155
- **Principal Investigator:** Dr. Jack Elston
- **Period of Performance:** September 29, 2025 – March 27, 2026 (6 months)
- **Project Status:** **ARCHIVED** — All work completed; final administrative tasks closed April 16, 2026

**Brief Description of Scope:**
Develop a modular, layered architecture for the SwiftCore avionics system enabling aircraft autonomy with multi-vehicle coordination and machine learning-based sensing and control. Key deliverables include: 5-layer functional architecture with pub/sub messaging protocol, ML module for terrain-following flight with automated PID tuning, safe sandbox framework, plugin architecture integration, supervisory controller with safety protocols, flight/bench test validation, and JSBSim simulation environment integration.
