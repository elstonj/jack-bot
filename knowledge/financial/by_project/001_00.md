# Project 001-00 — Financial Summary

## Project Overview
- **Full Project Name:** Coordinated UAS-Based Plume Tracking
- **Client/Agency:** Defense Threat Reduction Agency (DTRA)
- **Contract Type:** SBIR Phase I
- **Program/Solicitation:** DTRA252-001 PLUMES; Proposal Number T252-001-0052
- **Submission Date:** May 21, 2025
- **Period of Performance:** 7 months (proposed)
- **Brief Description:** Development of an AI-integrated, autonomous UAS system for real-time detection, tracking, and 3D volumetric reconstruction of airborne plumes during explosive tests. The system integrates Neural Radiance Fields (NeRFs) with BST's S2 UAS platform, SwiftCore™ Flight Management System, and multi-sensor data fusion capabilities. Phase I establishes technical feasibility through controlled laboratory experiments and field demonstrations.
