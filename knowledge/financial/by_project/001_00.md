## Project Overview
- **Full Project Name:** Coordinated UAS-Based Plume Tracking Technical Volume
- **Client/Agency:** Defense Threat Reduction Agency (DTRA)
- **Contract Type:** SBIR Phase I Proposal
- **Program/Solicitation:** DTRA 2025-001 PLUMES; Proposal Number T252-001-0052
- **Period of Performance:** 7 months (proposed in Phase I budget justification, dated 2025-05-21)
- **Brief Description:** AI-integrated autonomous UAS system for real-time detection, tracking, and 3D volumetric reconstruction of airborne plumes using Neural Radiance Fields (NeRFs) and dynamic scene reconstruction.
