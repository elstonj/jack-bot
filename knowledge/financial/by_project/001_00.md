# Project 001-00 — Financial Summary

## Project Overview
- **Full Project Name:** Coordinated UAS-Based Plume Tracking (DTRA PLUMES)
- **Client/Agency:** Defense Threat Reduction Agency (DTRA)
- **Contract Type:** SBIR Phase I Proposal
- **Program/Solicitation:** DTRA252-001 PLUMES; Proposal Number T252-001-0052
- **Period of Performance:** 7 months (proposal submitted May 21, 2025)
- **Brief Description of Scope:** Development of AI-integrated autonomous UAS system for real-time detection, tracking, and 3D volumetric reconstruction of airborne plumes during explosive tests, combining Neural Radiance Fields (NeRFs) and dynamic scene reconstruction algorithms with BST's S2 UAS platform and SwiftCore™ Flight Management System. Phase I establishes technical feasibility through controlled laboratory experiments and field demonstrations.
