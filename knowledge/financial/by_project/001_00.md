# Project 001-00 — Financial Summary

## Project Overview
- **Full Project Name:** Coordinated UAS-Based Plume Tracking
- **Client/Agency:** Defense Threat Reduction Agency (DTRA)
- **Contract Type:** SBIR Phase I
- **Program/Solicitation:** DTRA 252-001 PLUMES; Proposal Number T252-001-0052
- **Period of Performance:** 7 months (proposal submitted May 21, 2025)
- **Brief Description:** AI-integrated autonomous UAS system for real-time detection, tracking, and 3D volumetric reconstruction of airborne plumes from explosive tests using Neural Radiance Fields (NeRFs) and BST's fixed-wing UAS platforms integrated with SwiftCore™ Flight Management System. Phase I establishes technical feasibility through controlled laboratory experiments and field demonstrations.
