# Project 001-00 — Financial Summary

## Project Overview
- **Full Project Name:** Coordinated UAS-Based Plume Tracking (DTRA PLUMES)
- **Client/Agency:** Defense Threat Reduction Agency (DTRA)
- **Contract Type:** SBIR Phase I
- **Program/Solicitation:** DTRA252-001 PLUMES; Proposal Number T252-001-0052
- **Submission Date:** May 21, 2025
- **Period of Performance:** 7 months (proposed in budget justification)
- **Brief Description of Scope:** Development of AI-integrated autonomous UAS system for real-time detection, tracking, and 3D volumetric reconstruction of airborne plumes using Neural Radiance Fields (NeRFs) and BST's SwiftCore™ Flight Management System. Phase I focuses on technical feasibility through controlled laboratory experiments and field demonstrations. Work includes software development, flight planning implementation, sensor integration, and collaboration with Brookhaven National Laboratory.
