# Project 001-00 — Financial Summary

## Project Overview
**Project Name:** Coordinated UAS-Based Plume Tracking (DTRA PLUMES)  
**Client/Agency:** Defense Threat Reduction Agency (DTRA)  
**Contract Type:** SBIR Phase I  
**Program/Solicitation:** DTRA 2025-001 PLUMES; Proposal Number T252-001-0052  
**Submission Date:** May 21, 2025  
**Period of Performance:** 7 months (proposed)  

**Scope Description:**  
Development of an AI-integrated, autonomous UAS system to enable real-time detection, tracking, and 3D volumetric reconstruction of airborne plumes from explosive tests. The system combines Neural Radiance Fields (NeRFs) and dynamic scene reconstruction algorithms with BST's fixed-wing UAS platforms and SwiftCore™ Flight Management System. Phase I effort includes software development, flight planning implementation, sensor integration, controlled laboratory experiments, and field demonstrations, with collaboration from Brookhaven National Laboratory (BNL) for algorithm development.
