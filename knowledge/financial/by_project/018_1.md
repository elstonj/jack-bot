# Project 018-1 — Financial Summary

## Project Overview
- **Full Project Name:** Murphy's Pond CH4 Monitoring
- **Client/Agency:** Murray State University (Dr. Bassil El Masri, Procurement Services)
- **Contract Type:** Service contract for drone-based greenhouse gas measurement collection
- **Period of Performance:** 3 field campaigns planned for 2026 (May, July, August); original project due date 2025-01-31 (per Asana); flights extending through 2026; S2 rental due 2026-06-01
- **Brief Description of Scope:** Collection of CH₄, CO₂, and H₂O measurements over 5 km × 5 km area above Murphy's Pond, Hickman County, KY using Black Swift S2 UAS. Research focus: methane emissions from wetlands and bottomland hardwood forests; using remote sensing to upscale site measurements to larger geographic areas.
