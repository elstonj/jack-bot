# Project 025-04 — Financial Summary

## Project Overview
**Full Project Name:** Next-Generation Methane Monitoring for Orphaned Wells: Leveraging Drone Technology and Continuous Ground-Based Solution

**Client/Agency:** U.S. Department of Energy (DOE) Fossil Energy and Carbon Management (FECM)

**Contract Type:** DOE FECM program (specific solicitation number not provided in proposal)

**Proposal Submission Date:** 2025-04-09

**Brief Description of Scope:**
Integrated methane detection and monitoring system for orphaned well sites, combining Black Swift Technologies' S2 UAS platform equipped with Quanta3 methane sensor, eddy covariance flux towers with LI-COR monitoring systems, FluxMapper™ data analysis software, and HOO-nah remote connectivity device. Project aims to advance FluxMapper technology from TRL-4/5 to TRL-6, achieving 0.2 kg/h methane detection sensitivity. Three-phase development with Go/No-Go decision points.
